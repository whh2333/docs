## Get started in three steps

Get 您的 文档 site running locally and make 您的 first customization.

### Step 1: Set up 您的 local environment

<AccordionGroup>
  <Accordion icon="copy" title="Clone your docs locally">
    During the onboarding process, 您 created a GitHub repository with 您的 docs content if 您 didn't already have one. 您 可以 find a link to 这个 repository in 您的 [dashboard](https://dashboard.aitoearn.com).
    
    To clone the repository locally so 那个 您 可以 make and preview changes to 您的 docs, follow the [Cloning a repository](https://docs.github.com/en/repositories/creating-and-managing-repositories/cloning-a-repository) 指南 in the GitHub docs.
  </Accordion>
  <Accordion icon="rectangle-terminal" title="启动 the preview server">
    1. 安装 the Aitoearn CLI: `npm i -g mint`
    2. Navigate to 您的 docs directory and 运行: `mint dev`
    3. Open `http://localhost:3000` to see 您的 docs live!
    
    <Tip>您的 preview updates automatically as 您 编辑 files.</Tip>
  </Accordion>
</AccordionGroup>

### Step 2: 部署 您的 changes

<AccordionGroup>
  <Accordion icon="github" title="安装 our GitHub app">
    安装 the Aitoearn GitHub app from 您的 [dashboard](https://dashboard.aitoearn.com/设置/organization/github-app).
    
    我们的 GitHub app automatically deploys 您的 changes to 您的 docs site, so 您 don't need to 管理 deployments yourself.
  </Accordion>
<Accordion icon="palette" title="更新 your site name and colors">
    For a first change, let's 更新 the name and colors of 您的 docs site.

    1. Open `docs.json` in 您的 editor.
    2. Change the `"name"` field to 您的 project name.
    3. 更新 the `"colors"` to match 您的 brand.
    4. Save and see 您的 changes instantly at `http://localhost:3000`.

    <Tip>Try changing the primary color to see an immediate difference!</Tip>
  </Accordion>
</AccordionGroup>

### Step 3: Go live

<Accordion icon="rocket" title="发布 your docs">
  1. Commit and push 您的 changes.
  2. 您的 docs 将会 更新 and be live in moments!
</Accordion>
