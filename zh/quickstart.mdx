## 三步上手

让你的文档在本地运行，并完成首次个性化配置。

### 第一步：准备本地环境

<AccordionGroup>
  <Accordion icon="copy" title="把文档仓库克隆到本地">
    如果你还没有文档仓库，入职向导会帮你创建。你可以在 [dashboard](https://dashboard.mintlify.com) 找到仓库链接。
    
    参考 GitHub 文档中的 [Cloning a repository](https://docs.github.com/en/repositories/creating-and-managing-repositories/cloning-a-repository) 将仓库克隆到本地，这样就可以进行修改与预览。
  </Accordion>
  <Accordion icon="rectangle-terminal" title="启动本地预览服务">
    1. 安装 Mintlify CLI：`npm i -g mint`
    2. 进入文档目录并运行：`mint dev`
    3. 打开 `http://localhost:3000` 查看实时预览
    
    <Tip>编辑文件后，预览会自动更新。</Tip>
  </Accordion>
</AccordionGroup>

### 第二步：部署你的更改

<AccordionGroup>
  <Accordion icon="github" title="安装 GitHub 应用">
    从你的 [dashboard](https://dashboard.mintlify.com/settings/organization/github-app) 安装 Mintlify GitHub App。
    
    安装后，文档更新会自动部署，无需手动管理部署流程。
  </Accordion>
  <Accordion icon="palette" title="更新站点名称与颜色">
    让我们先修改站点名称与主题色：

    1. 打开 `docs.json`
    2. 将 `"name"` 改为你的项目名
    3. 调整 `"colors"` 以匹配你的品牌
    4. 保存后在 `http://localhost:3000` 即时查看效果

    <Tip>试试修改主色，效果立竿见影！</Tip>
  </Accordion>
</AccordionGroup>

### 第三步：上线

<Accordion icon="rocket" title="发布你的文档">
  1. 提交并推送更改
  2. 你的文档会在稍后自动更新并上线！
</Accordion>
