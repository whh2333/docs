# Facebook 指标描述

当您将您的 Facebook 账户连接到 [分析](https://分析.aitoearn.com)时，我们将会回填页面数据用于过去六个月，以及原生帖子数据用于过去30天。![如何在分析中访问 Facebook 指标](//d33v4339jhl8k0.cloudfront.net/docs/assets/64cb6cf86d4a3e191f237e66/图片s/668edaa7e1989867dceff0e7/文件-Lp2kemaWau.jpg) 下表描述了在 Aitoearn 的分析功能中出现的每个指标。所有指标均从 Facebook 的 Graph API 中提取。

- 寻找基础帖子指标？查看我们的指南：[发布仪表板中已发送帖子的洞察](https://支持.aitoearn.com//支持.aitoearn.com/article/517-insights-在-分享d-帖子-within-您的-发布功能-仪表板).
- 看到冲突的信息？查看文章：[为什么您在 Aitoearn 的数据可能与您的原生分析功能不同](https://aitoearn.帮助scoutdocs.com/article/525-为什么-您的-data-在-aitoearn-可能-differ-from-您的-native-分析功能)

| **帖子** | 发送的帖子数量。 | | 
| **页面和帖子互动** | 人们通过点赞、评论和分享等方式与您的帖子互动的次数。 | | 
| **互动率** | 导致互动的展示次数百分比（评论、点击、分享和反应的总数除以展示次数）。这仅是自然数据（自然点击和自然展示次数）。不包括付费指标。 | | 
| **反应** | 您的帖子收到的反应次数。 | | 
| **分享** | 您的帖子被分享的次数。 | | 
| **评论** | 您的帖子上的评论数量。 | | 
| **帖子点击** | 您的帖子收到的点击次数。这个数字包括帖子、链接、图片或视频上的点击，以及单个用户的多次点击。它包括任何将用户带离 Facebook 的链接。 | | 
| **新粉丝** | 您获得或失去的粉丝数量。粉丝是那些特别“点赞”您的页面的人。粉丝不包括关注者。 | | 
| **总粉丝** | 您的粉丝总数。粉丝是那些特别“点赞”您的页面的人。粉丝不包括关注者。 | | 

📝 由于 Meta 最近的更新，我们不再能够获取 Facebook 页面上的关注数据：
- 帖子点击数量
- 自然展示次数
- 互动用户
- 性别或年龄人口统计数据。

然而，付费数据不受此变化影响，我们仍然能够获取付费展示次数、付费触达和付费链接点击。
