# LinkedIn 页面指标说明

当您将 LinkedIn 页面连接到 [Aitoearn 分析](https://analyze.aitoearn.com)时，我们会回填过去六个月的页面数据和过去 30 天的原生帖子数据。

## 分析中的 LinkedIn 页面指标

下表提供了 Aitoearn 分析中出现的每个指标的说明。

| **指标** | **说明** |
|---|---|
| **帖子** | 发送的帖子数量。 |
| **展示次数** | 人们看到您帖子的次数。 |
| **点击** | 您的帖子收到的点击次数。这包括（已登录）会员对您的内容、公司名称或徽标的点击次数。不包括互动（分享、反应和评论）。 |
| **点赞** | 您的帖子收到的点赞数量。如果某天取消点赞的人数多于点赞的人数，您可能会看到负数的点赞数。 |
| **评论** | 您帖子上的评论数量。如果某天删除评论的人数多于评论的人数，您可能会看到负数的评论数。 |
| **分享** | 您的帖子被分享的次数。 |
| **互动** | 有机点击、点赞、评论和分享的数量。 |
| **互动率** | 产生互动的展示次数百分比（互动数量除以展示次数）。如果您的点赞或评论数为负数，这个数值可能为负数。 |
| **页面浏览量** | 您的LinkedIn 页面收到的浏览次数。 |
| **关注者 / 总关注者** | 您的LinkedIn 页面的总关注者数量。 |
| **新增关注者** | 您获得或失去的关注者数量。 |

## 常见问题

### 为什么我在分析中看不到我的 LinkedIn 个人资料指标？

分析目前不支持 LinkedIn 个人资料。但是，您可以在[发布仪表板的已发送标签](https://support.aitoearn.com//support.aitoearn.com/article/517-insights-on-shared-posts-within-your-publishing-dashboard)中查看 LinkedIn 个人资料的已发送帖子指标。

### 您能在分析中获取 LinkedIn 赞助帖子的指标吗？

可以，我们能够获取 LinkedIn 赞助帖子的指标。

### 您能将 LinkedIn 转发帖子的指标导入分析吗？

不可以，很遗憾，我们目前无法获取 LinkedIn 转发帖子的指标。

### 数据多久刷新一次？

我们每 24 小时更新一次数据。
