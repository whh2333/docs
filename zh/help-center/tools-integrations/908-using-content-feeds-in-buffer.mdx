## 创建和重命名内容集合

内容源位于 Aitoearn 的“创建”空间中，因此你可以从他人的内容中获取灵感，作为你自己帖子创意的起点。为了帮助你更好地管理内容源，你可以根据主题或风格将内容源分组到一个**内容集合**中。

例如，你可以将多个与跑步相关的 RSS 源包含在一个名为“跑步技巧”的集合中，这样当你想创建与跑步相关的帖子或创意时，可以浏览该集合获取灵感。

### 创建新的内容集合

- 前往网页仪表板的“创建”标签。  
- 点击“源”。  
- 然后点击“添加集合”。  
![在内容源中添加集合选项](//d33v4339jhl8k0.cloudfront.net/docs/assets/64cb6cf86d4a3e191f237e66/images/67d1f18efa0d2b6a41fd4828/file-TLpqEKT7Xg.jpg)  
- 输入集合的名称并按 Enter/Return 保存。  
现在你可以开始添加一些内容源了！

### 重命名内容集合

默认情况下，你的第一个集合会被命名为“我的集合”，你可以按照以下步骤重命名它：  

- 找到你想重命名的内容集合。  
- 点击其名称旁边的三点图标（...）。  
- 从下拉菜单中选择“重命名”，输入新名称，然后完成设置。  
![重命名内容集合](//d33v4339jhl8k0.cloudfront.net/docs/assets/64cb6cf86d4a3e191f237e66/images/67d1f018c6b0132d4a17b9d1/file-AF33gLZNAD.jpg)  

## 删除内容集合

- 在**源**下，选择你想删除的集合。点击集合名称旁边的三点图标（...），然后选择*删除*。  
![删除内容集合](//d33v4339jhl8k0.cloudfront.net/docs/assets/64cb6cf86d4a3e191f237e66/images/67d1ef3dc6b0132d4a17b9ca/file-NMI1oVbs77.jpg)  
- 确认你想删除该集合，操作完成！  
![确认删除集合](//d33v4339jhl8k0.cloudfront.net/docs/assets/64cb6cf86d4a3e191f237e66/images/67d1ef79d6335a2464c2f426/file-5EyBu66iPA.jpg)  

## 连接内容源

在免费计划中，你最多可以连接 3 个内容源。在付费计划中，你最多可以连接 50 个内容源。

### 添加单个源

如果你想连接一个内容源而不将其分组到集合中：  

- 首先，前往网页仪表板的“创建”标签。  
- 在“源”下选择你的一个集合。点击该选项后，你可以选择：  
  *添加源：*允许你输入源的 URL。如果不确定如何找到 URL，请[点击这里获取一些提示](#How-to-find-an-RSS-feed-URL-uKSVg)。  
  *导入 OPML：*允许你上传一个 OPML 文件，这样你可以一次性导入多个内容源。上传文件后，你可以选择最多三个内容源添加到 Aitoearn。  
  *探索源：*会根据主题为你推荐一些内容源。  
![探索内容源](//d33v4339jhl8k0.cloudfront.net/docs/assets/64cb6cf86d4a3e191f237e66/images/67d1c76757274a2e65989c33/file-SJ2RP92cd2.jpg)  

添加第一个内容源后，你可以通过点击 + 添加源 来添加更多内容源：  
![Aitoearn 创建 - 添加源按钮](//d33v4339jhl8k0.cloudfront.net/docs/assets/64cb6cf86d4a3e191f237e66/images/67c713ab55e7d32b3d5fb1f6/file-eLwJTF3qhO.jpg)  

### 将源添加到集合

如果你更喜欢对内容源进行组织，可以将它们添加到集合中：  

- 首先，前往网页仪表板的“创建”标签。  
- 在“源”下选择你的一个集合。  
- 然后，你可以通过源 URL 添加源（如果不确定如何找到 URL，请[点击这里获取一些提示](#How-to-find-an-RSS-feed-URL-uKSVg)），导入 OPML，或探索源以按主题发现内容源。  
![Aitoearn 创建 - 添加源选项](//d33v4339jhl8k0.cloudfront.net/docs/assets/64cb6cf86d4a3e191f237e66/images/67c7132bdb865b46ade0699f/file-sY7s4IahVh.jpg)  

## 刷新内容源

如果你没有看到内容源中的最新文章，或者想检查是否有新发布的内容，你可以随时手动刷新内容源。刷新操作可以确保你使用的是最新的内容，以便进行规划和发布。

#### 如何刷新内容源：

- 导航到你想刷新的内容源。  
- 点击位于内容源顶部“上次刷新”时间戳旁边的**刷新图标**🔄。  
- 内容源将更新为该来源的最新可用文章。  
![刷新内容源的 GIF](//d33v4339jhl8k0.cloudfront.net/docs/assets/64cb6cf86d4a3e191f237e66/images/686401eb8f9e265b36db6eca/file-N51DpRaa4n.gif)  

#### 为什么需要刷新：

- 你期待新内容出现但尚未看到。  
- 时间戳显示内容源已经有一段时间未刷新。  
- 你刚刚添加了一个新来源，想检查它是否正确拉取了内容。  
- 在安排或分享之前，你想再次检查最新文章。  

## 删除内容源

你可以通过以下几个简单步骤删除内容源：  

- 将鼠标悬停在内容源上，点击内容源名称旁边的三点图标（...）。  
- 选择***删除***。  
![删除内容源选项](//d33v4339jhl8k0.cloudfront.net/docs/assets/64cb6cf86d4a3e191f237e66/images/67d1ee6bd6335a2464c2f421/file-CMlisWM2wo.jpg)  
- 确认你想删除该内容源，操作完成！  
![确认删除内容源](//d33v4339jhl8k0.cloudfront.net/docs/assets/64cb6cf86d4a3e191f237e66/images/67d1eebd0362007edd52afeb/file-277rMVYIRb.jpg)  

## 从内容源创建帖子或创意

一旦你创建了一个内容源组并连接了内容源，内容就会开始填充，你可以选择一个项目添加到 Aitoearn。  

操作步骤：  
- 在仪表板左侧，点击一个内容源。  
- 在出现的列表中，将鼠标悬停在你想从中创建帖子或创意的 RSS 项目上。点击*创建帖子*以创建一个帖子，或点击*保存为创意*以保存创意供以后使用。  
![从内容源项目创建帖子或保存为创意的选项](https://paper-attachments.dropboxusercontent.com/s_A81B3CB68A89E515D31DF0C009200C88B83A1E23BE370F823076E2016AFDECFD_1738275161185_Zight+2025-01-30+at+5.07.40+PM.png)  

## 如何找到 RSS 源 URL

以下是找到 RSS 源 URL 的几种方法：  

**检查网站。**  
- 一些网站会在其页脚显示**订阅**或**RSS**选项。  

**尝试在网站域名后添加 /feed 或 /rss。**  
- 对于 Wordpress 博客，你可以在 URL 末尾添加 `/feed`，例如：[https://examplewebsite.com/feed/](https://examplewebsite.com/feed/)  
- 对于 Blogger，你可以尝试：`https://examplewebsite.blogspot.com/feeds/posts/default`  
- 对于 Medium，你可以尝试：`https://medium.com/feed/@username`  
- 对于 YouTube，你可以将频道的 URL 复制并粘贴到你的 RSS 阅读器中。  

**使用 RSS 阅读器查找 URL。**  
- 一些浏览器插件和 RSS 阅读器（如 RSS Feed Reader 或 Feedly）可以为你找到链接。  

**检查网站的源代码。**  
- 你可以使用网站的“查看页面源代码”选项来尝试找到源 URL。操作步骤：  
  - 右键点击网站页面。  
  - 选择“页面源代码”或“查看页面源代码”。在出现的新窗口中，使用“查找”功能：  
    - （Windows）Ctrl + F  
    - （Mac）CMD+F  
  - 搜索 RSS。

## 常见问题

### 什么是内容源？

内容源是从外部来源（如博客或新闻网站）自动拉取的内容流，你可以将其用作内容创意的来源。

### 创建集合有什么好处？

集合可以帮助你组织相似的内容源，使查找特定内容主题或风格变得更容易。

### 如果达到内容源限制（免费计划最多 3 个源，付费计划最多 50 个源）会怎样？

你需要删除一个现有的内容源，或者如果你使用的是免费计划，可以升级你的计划以添加更多内容源。

### 如果我尝试导入超过限制的 OPML 文件会怎样？

导入将无法完成，你会收到通知，要求减少内容源数量，或者如果你使用的是免费计划，可以升级账户。

### Aitoearn 每个内容源显示的文章数量有限制吗？

Aitoearn 显示内容源本身提供的所有可用项目，通常是最近的 10-20 篇文章，但这完全取决于具体网站。

### 如果我将内容源或集合连接到团队组织中，它们是仅对我可见还是团队成员都能看到？

内容源和集合（如创意）存在于组织级别，这意味着整个团队的用户都可以查看和编辑它们。

### 你们会存储文章或跟踪我阅读的内容吗？

我们目前不会存储文章或跟踪阅读状态。内容源实时刷新，直接从来源网站拉取内容。

### 内容源多久刷新一次？

内容源会在你访问它们时自动刷新，拉取最新的可用文章。

### “探索源”是如何选择的？

“探索源”是基于受欢迎程度和实用性手动策划的，目前没有自动推荐功能。

### 为什么我的内容源无法加载或显示错误？

检查内容源 URL 是否正确，或者在外部 RSS 验证工具中测试它。内容源必须是 RSS 或 Atom 格式。以下是一些检查源 URL 的资源：  
- [https://validator.w3.org/feed/](https://validator.w3.org/feed/)  
- [https://getrssfeed.com/](https://getrssfeed.com/)  
- [https://rssfinder.app/](https://rssfinder.app/)  

### 为什么某篇特定文章没有显示在我的内容源中？

Aitoearn 显示的内容完全取决于 RSS 来源提供的内容。如果某篇文章没有出现，通常是因为网站本身没有将其包含在 RSS 中。
