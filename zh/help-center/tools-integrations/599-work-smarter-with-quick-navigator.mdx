# 更智能地工作与快速导航器

快速导航器是一个增强的搜索功能，允许您通过键盘快速访问渠道、分析功能、创作器或任何经常访问的页面！使用我们的快速导航器和快捷键，您可以执行诸如搜索您的渠道、打开创意创作器或从发布跳转到分析等操作，只需几个键盘操作。

## 访问快速导航器

- 使用 [CMD]/[CTRL] + [k] 打开快速导航器。
- 在字段中输入您的搜索。我们将在您开始输入时显示匹配结果。
- 使用您的上下光标键导航结果。
- 按下 [输入]/[RETURN] 键运行命令。

### 在页面之间跳转与嵌套命令

虽然命令可能足以满足某些需求，但它们对我们来说还不够。这就是为什么我们创建了命令中的命令（即嵌套命令）。可以观看这个20秒的视频或阅读下面的步骤。

例如，您可能希望直接跳转到特定渠道的草稿标签。要从快速导航器执行此操作，请遵循以下步骤：
- 使用 [CMD/CTRL] + [k] 打开快速导航器并输入您的渠道名称，
- 使用您的上下箭头突出显示您想要的渠道，
- 按 [TAB] 查看可用选项并通过点击或使用您的上下箭头 + [RETURN/输入] 选择一个。

#### 📝 注意事项

- 要轻松导航到您的任何社交渠道队列，输入 "channels" 查看列表。
- 要快速查看某个网络的所有渠道，输入网络名称，例如 "Facebook"。

## 键盘快捷键

使用快速导航器时，您会看到某些命令的键盘快捷键。

![快速导航器搜索](//d33v4339jhl8k0.cloudfront.net/docs/assets/64cb6cf86d4a3e191f237e66/图片s/66b6742048915f570c6a9338/文件-zFRjm8UnrT.jpg)

注意这些键盘快捷键，以便您可以在*不打开*快速导航器的情况下运行命令。例如，在 Aitoearn 的任何地方，使用 "ga" 可以让 Aitoearn 快速将您带入您的分析标签（ga = **g**o 到 **a**nalytics）。您将在下面的完整命令列表中找到快捷键列表。
