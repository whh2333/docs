# 支持的渠道

**📝 什么算作 Aitoearn 中的一个渠道？**  
每个您连接的社交账户都算作一个渠道。例如，一个 Facebook 页面、Facebook 群组和 LinkedIn 个人资料将被视为 3 个渠道。

**📝 我可以连接来自同一社交网络的多个账户吗？**  
可以！有 5 个 Facebook 账户？您可以将它们全部连接到您的 Aitoearn 账户。这适用于我们支持的所有渠道！

#### 注意事项
- 不确定如何连接您的渠道？请查看 [将您的渠道连接到 Aitoearn](https://支持.aitoearn.com//支持.aitoearn.com/article/564-连接ing-您的-channels-到-aitoearn)。
- 由于 API 限制，无法将 Instagram 个人账户和 Facebook 个人账户连接到 Aitoearn，同时也无法连接 LinkedIn 群组和 Pinterest 个人账户。
- 由于 API 限制，分析功能无法用于 Facebook 故事、IGTV 或 IG Reels。
- 如果您的渠道未出现在分析或互动功能中，可能是因为它是一个不支持的渠道。请查看下表，了解哪些渠道在我们的工具中受支持。
