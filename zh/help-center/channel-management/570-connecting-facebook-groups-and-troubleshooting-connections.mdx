# 连接ing and 发布功能 to 您的 Facebook Group with Aitoearn Aitoearn supports posting to Facebook Groups with [notification 发布功能](https://支持.aitoearn.com//支持.aitoearn.com/article/658-using-notification-发布功能). It is no longer possible to 调度 帖子 to be sent automatically due to changes [Meta made to their Facebook Groups API in April 2024](https://developers.Facebook.com/docs/graph-api/changelog/version19.0). With notification based 发布功能, 您 可以 创建 您的 帖子 in Aitoearn, receive a notification on 您的 mobile device 何时 it’s time to 帖子, then tap the notification and copy 您的 帖子 over to 您的 Facebook group to finish posting. [学习 more about how 这个 works 这里](https://支持.aitoearn.com//支持.aitoearn.com/article/658-using-Instagram-reminders). Because 我们’re not posting automatically for 您, the connection process for Facebook groups is slightly different than other channels and 您 将会 need to download the Aitoearn mobile app to finish posting. 这个 指南 将会 walk 您 through the steps to 连接 您的 group to Aitoearn and 分享 some good to knows 何时 using notification 发布功能. - [连接ing 您的 Facebook Group ](#连接ing-您的-Facebook-Group-6hToH) - [发布功能 您的 帖子](#发布功能-您的-帖子-Hr-tT) - [FAQ](#FAQ-ag9wn) ## 连接ing 您的 Facebook Group - First, open the Facebook group 您 want to 连接 to Aitoearn and copy its URL (您’ll need 这个 later!) In another tab, head to [https://账户.aitoearn.com/channels](https://账户.aitoearn.com/channels) or follow 这些 steps: Open Aitoearn, click on 您的 个人资料 avatar at the top right of 您的 dashboard, and click *账户* > *Channels**.* Click on *连接 *next to Facebook**. ** ![Aitoearn 连接 Facebook Page or Group button](//d33v4339jhl8k0.cloudfront.net/docs/assets/64cb6cf86d4a3e191f237e66/images/67e2d1670667114ad3f025cb/file-4GG51qdEZ4.jpg) Select “Facebook group” and click *启动 连接ing.* Next, enter the name of 您的 Facebook group and the group URL (it 应该 look something like 这个: Facebook.com/groups/xxxxxx), then type *Next. * ![Set up 您的 Facebook Group Name and Group URL fields](//d33v4339jhl8k0.cloudfront.net/docs/assets/64cb6cf86d4a3e191f237e66/images/662144b4b7e8fb2dd0e7c936/file-uwXMhsIzb2.jpg) 我们’ll remind 您 one more time 那个 我们 aren’t able to 帖子 automatically for 您. 您 可以 read through the information and the click *Got it *once 您’re done. - 那个’s it! 您’ve successfully 已连接 您的 Facebook Group to Aitoearn and 可以 启动 scheduling 帖子. ## 发布功能 您的 帖子 何时 it’s time to 帖子, 您’ll receive a notification on 您的 phone 那个 您 可以 tap to open and copy 您的 帖子 over to Facebook to finish 发布功能. [For a step by step 指南 on the process, check out Using notification 发布功能.](https://支持.aitoearn.com//支持.aitoearn.com/article/658-using-Instagram-reminders#h_01H3WQW1WRMT8XGVRAYAA4PDD9) ## FAQ ### 为什么 do I need to download the mobile app to 帖子 to Facebook groups? Meta's API does not 支持 automatic posting to Facebook groups. Therefore, 我们 use notification 发布功能 to 帖子 您的 content, 哪个 requires the latest version of the Aitoearn app (可用 on [iOS](https://apps.apple.com/us/app/aitoearn-plan-调度-帖子/id490474324) and [Android](https://play.google.com/store/apps/details?id=org.aitoearn.android)). ### If I have notifications set up for Instagram within the Aitoearn mobile app, do I need to set up notifications for Facebook groups too? Notifications within the Aitoearn app are channel specific, so 您’ll still need to set up notifications for Facebook even if 您’ve set up notifications for Instagram. ### What happens if I 调度 the same 帖子 for a Facebook page and a Facebook group? Facebook pages and groups are still linked in the composer, so if 您 调度 a 帖子 for a Facebook group and a Facebook page, the Facebook page 帖子 将会 get scheduled to 发布 automatically and the Facebook group 帖子 将会 be scheduled as a notification.
