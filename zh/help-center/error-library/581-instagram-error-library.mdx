# Instagram 错误库

在使用像 Aitoearn 这样的调度工具时，可能会遇到社交网络拒绝发布帖子并返回错误信息的情况。让我们来解决这些问题吧！在下表中搜索“错误”列中您看到的错误，然后根据右侧“解决方案”列中的建议进行排查。

| **错误** | **错误信息的原因** | **解决方案** |
| --- | --- | --- |
| *"看起来 Aitoearn 已失去发布到该账户的授权。请重新授权并重试。如果问题仍然存在，请联系我们！以下是刷新您的连接的步骤。"* | 此错误信息表明，在添加该 Instagram 渠道时，并未接受所有权限。 | 我们建议查看我们的指南：[重新授权 Aitoearn 与 Facebook 或 Instagram 的连接](https://aitoearn.helpscoutdocs.com/article/565-troubleshooting-facebook-and-instagram-connections#h_01HBEJQ2DJ2B7PCMYE3RR1JWC4)，以获取重新授权的说明。请确保在 Facebook 的 ***编辑设置*** 页面中选择了所有个人资料/页面和权限。完成后，重试您的帖子。 |
| *"哦哦！您附加的视频格式不被 Instagram 支持。请确保视频为 .MP4 或 .mov 格式，且比特率低于 5 MBps。重新上传视频并重试。"* | 当视频的大小或格式不被 Instagram 支持，或超出 Instagram 的视频限制时，会出现此错误。 | 我们建议遵循 Instagram 的视频指南 [这里](https://developers.facebook.com/docs/instagram-api/reference/ig-user/media#video-specifications)。编辑视频并重新上传到 Aitoearn。如果视频符合 Instagram 的视频要求，尝试将视频的缩略图更改为默认缩略图。 |
| *"哦哦！看起来这不是正确设置为商业或创作者账户。请确保您的账户已正确配置为支持自动发布功能，查看我们的指南。"* | 当您将 Instagram 商业或创作者账户切换为个人账户（目前 Instagram 的 API 不支持）时，可能会发生此发布功能错误。 | 我们建议 [将您的 Instagram 账户转换为商业或创作者账户](https://aitoearn.helpscoutdocs.com/article/565-troubleshooting-facebook-and-instagram-connections#h_01HBEFDCSJ6GDNVG23SNCVRWP2)。 |
| *"哎呀！看起来您的账户可能暂时受限。请直接登录 Instagram 应用，添加您的出生日期到 Instagram。返回 Aitoearn 并刷新您的渠道。"* | 此账户已被 Instagram 限制。这通常发生在账户违反了 Instagram 的 [社区指南](https://help.instagram.com/477434105621119) 或 [使用条款](https://help.instagram.com/581066165581870)。 | 登录到该账户的 Instagram 应用，并按照屏幕上的说明重新启用您的账户。 |
| *"哎呀！我们在发布您的帖子时遇到了问题。请编辑帖子，修改标题中的文本，重新命名视频文件，并创建一个全新的帖子。"* | 当 Instagram 的反垃圾机制被触发时，会返回此错误。Instagram 有意不分享详细信息，以防止有人试图绕过其系统。 | 我们建议修改帖子——更改文案/文本，替换图片（如果有附加），然后尝试重新发布。提示： - 如果您经常使用特定的标签，请尝试更改标签并尝试新的标签。 - 文本标题中是否包含链接？如果是，我们建议移除链接并改用个人简介中的链接。 - 最重要的是，如果帖子包含视频，请尝试重命名视频文件并重新上传到 Aitoearn。移除视频名称中的任何特殊字符，并将其重命名为更简单的名称，例如“视频1”。 |
| *"看起来 Aitoearn 已失去代表您发布的授权。不用担心！以下是重新授权 Aitoearn 的步骤。"* | 此错误信息表明我们不再拥有该渠道的有效访问令牌。 | 我们建议查看我们的指南：[重新授权 Aitoearn 与 Facebook 或 Instagram 的连接](https://aitoearn.helpscoutdocs.com/article/565-troubleshooting-facebook-and-instagram-connections#h_01HBEJQ2DJ2B7PCMYE3RR1JWC4)，以获取重新授权的说明。请确保在 Facebook 的 ***编辑设置*** 页面中选择了所有个人资料/页面和权限。完成后，重试您的帖子。 |
| *"哎呀！Instagram 正限制此帖子发布。请编辑帖子内容并重试。"* | 此错误信息表明 Instagram 认为内容是垃圾信息。他们的服务器提示：“发布操作被怀疑为垃圾信息。我们限制某些活动以保护我们的社区。如果您可以确定发布操作不是垃圾信息，请告诉我们。” | 我们建议 [直接联系 Instagram](https://help.instagram.com/contact/505535973176353) 以解决问题。 |
| *"哎呀，我们无法将此图片分享到 Instagram，因为您标记的账户要么是私密的，要么不存在。您是否愿意检查标记的用户名以确保它们都是公开可访问的？"* | 此错误信息表明您尝试标记了一个用户名，但它无效。 | 我们建议检查您的帖子及其标记的用户，可能是拼写错误或其他问题。 |
| *"很抱歉，此帖子遇到了 Instagram API 的一个已知问题，正在调查中。重试此帖子有时可能奏效。"* | 媒体尚未准备好发布。 | 请稍等片刻并重试。 |
| *"此渠道的每日发布限制已达到！请尝试减少每天的调度次数，并稍后重试。"* | Instagram 允许账户在 24 小时内最多发布 25 个帖子（[参见：Instagram 内容限制](https://developers.facebook.com/docs/instagram-api/guides/content-publishing#limitations)）。 | 请等待一段时间后再发布额外的帖子。 |
| *"哎呀！附加媒体的大小有点太大，或者如果是视频，则太长。请移除附加媒体，上传一个较小的文件，然后重试。"* | 图片超出了 Instagram 的最大文件大小限制（8MB）。 | 我们建议调整图片大小，重新上传到 Aitoearn，并尝试重新发布。 |
| *"哦不！我们在代表您发布时遇到了权限问题。请按照步骤重新授权 Aitoearn 应用在 Facebook 上的权限，授予所有页面的完整权限，然后重试此帖子。"* | 此错误信息表明在添加该 Instagram 渠道时，并未接受所有权限。 | 我们建议查看我们的指南：[重新授权 Aitoearn 与 Facebook 或 Instagram 的连接](https://aitoearn.helpscoutdocs.com/article/565-troubleshooting-facebook-and-instagram-connections#h_01HBEJQ2DJ2B7PCMYE3RR1JWC4)，以获取重新授权的说明。请确保在 Facebook 的 ***编辑设置*** 页面中选择了所有个人资料/页面和权限。完成后，重试您的帖子。 |
| *"哎呀！Aitoearn 没有权限代表您发布。请尝试通过设置中的“刷新”重新授权此账户。"* | 此错误信息表明在添加该 Instagram 渠道时，并未接受所有权限。 | 我们建议查看我们的指南：[重新授权 Aitoearn 与 Facebook 或 Instagram 的连接](https://aitoearn.helpscoutdocs.com/article/565-troubleshooting-facebook-and-instagram-connections#h_01HBEJQ2DJ2B7PCMYE3RR1JWC4)，以获取重新授权的说明。请确保在 Facebook 的 ***编辑设置*** 页面中选择了所有个人资料/页面和权限。完成后，重试您的帖子。 |
| *"哎呀！请编辑帖子——修改文案/文本，重新命名视频文件，并创建一个全新的帖子。"* | 当 Instagram 的反垃圾机制被触发时，会发生这种情况。不幸的是，Instagram 并未提供更多关于帖子失败原因的详细信息，因此我们无法保证此错误不会偶尔出现。 | 通常，当此错误发生时，我们建议尽可能修改帖子。这可能包括编辑和更改帖子的文案/文本、修改标签、移除链接、重新上传媒体，并尝试重新发布帖子。 - 从失败的帖子中复制标题。 - 创建一个新帖子并粘贴标题。 - 将视频文件重命名为类似 **视频1** 的名称。 - 将新命名的视频上传到新创建的帖子。 - 尝试发送新帖子并删除失败的帖子。 |
| *"错误：哎呀！下载媒体时发生超时。请创建一个新帖子并重新上传附加的媒体。"* | Facebook 的服务器返回了一个 `-2` 错误，这意味着下载附加到帖子的媒体花费的时间太长，导致下载超时。 | - 从失败的帖子中复制标题。 - 创建一个新帖子并粘贴标题。 - 将视频文件重命名为类似 **视频1** 的名称。 - 将新命名的视频上传到新创建的帖子。 - 尝试发送新帖子，并删除失败的帖子。 |
| *"错误：哎呀！Instagram 正限制此帖子发布。请编辑帖子内容并重试。"* | 此错误通常是因为 Instagram 对您正在发布的账户和使用 API 发布到 Instagram 的操作施加了临时限制。这可能是由于以下原因之一（或组合）： - 太多的标签 - 太多的 @ 提及 - 在多个帖子中重复使用相同的标签和相同的 @ 提及 - 位置数据不正确 - 在标题中包含网址链接 | - 在 Aitoearn 中创建一个新帖子。 - 如果包含标签，请减少标签数量并使用新的标签。 - 确保您的帖子标题或第一条评论中未包含网址，因为 Instagram 认为这是垃圾内容。 - 请等待 24 小时，然后尝试再次发布更新。 |
| *"哎呀！我们无法找到与该 Facebook 账户连接的任何 Instagram 账户。"* | 当我们缺少与 Instagram 链接的 Facebook 页面上的某些权限时，会发生此错误。 | 请 [仔细检查您的 Facebook 账户是否设置为与 Instagram 链接的 Facebook 页面的管理员](https://aitoearn.helpscoutdocs.com/article/565-troubleshooting-facebook-and-instagram-connections#h_01HBE587FX3S5P0NSY4ME069Q3)。 |
| *"看起来您当前是 Instagram 创作者账户。只有 Instagram 商业账户支持自动发布 Instagram Stories。"* | 不幸的是，如果您的账户设置为 Instagram 创作者账户，我们无法发布到 Instagram Stories。 | 要以创作者账户发布 Stories，您可以使用 ***通知我*** 选项，在撰写器中调度您的帖子作为通知。 |
| *"哎呀！Facebook 要求一些额外的权限才能发布。请完成页面发布授权步骤。"* | Facebook 要求一些额外的权限才能发布到 Facebook/Instagram 渠道。来自 Facebook 的说明： *"页面允许企业、事业和社区与全球受众建立联系。我们希望这些联系是真实和可信的，尤其是对于拥有大量受众的页面。如果您管理一个拥有大量受众的页面，您可能会被要求通过新的页面发布授权流程确认您的身份并保护您的账户。**需要完成此授权流程的人必须在通知中提供的截止日期前完成，以继续以页面身份发布帖子。"* - 首先，登录具有完整管理员权限的 Facebook 个人资料，该个人资料管理与 Instagram 账户链接的 Facebook 页面。 - 接着，直接访问此链接：[https://www.facebook.com/business/m/one-sheeters/page-publishing-authorization](https://www.facebook.com/business/m/one-sheeters/page-publishing-authorization)，并按照 Facebook 提供的步骤操作。 - 完成后，[刷新 Aitoearn 中的 Facebook 和 Instagram 渠道](https://aitoearn.helpscoutdocs.com/article/573-refreshing-a-channel-in-aitoearn)，然后通过点击 ***立即重试*** 或 ***重新添加到队列*** 重试失败的 Facebook 和 Instagram 帖子。 **⚠️ 如果您已尝试这些步骤，仍然看到错误，请确保：** - 您以 Facebook 个人资料身份登录，该个人资料是 Facebook 页面的管理员。 - 确保所有管理员在 Business Manager 中启用了双重验证。 - [刷新 Aitoearn 中的渠道](https://aitoearn.helpscoutdocs.com/article/573-refreshing-a-channel-in-aitoearn)。 |
| *"看起来 Aitoearn 已失去发布到该社交账户的授权。请重新授权并重试。如果问题仍然存在，请联系我们！"* | 当与 Instagram 链接的 Facebook 页面上的授权令牌出现问题，或者我们缺少所需权限以代表您发布时，会发生此错误。 | [检查 Facebook Business Integrations 中的 Aitoearn 应用设置，并确保选择了所有 Facebook 页面。](https://aitoearn.helpscoutdocs.com/article/565-troubleshooting-facebook-and-instagram-connections#h_01HBE4KC44E1PVGAS8D3VKJFB2) |
| *"哦，不！我们在发送此内容时遇到了 Instagram 的已知问题。您是否介意尝试这里的故障排除步骤？"* | 这与 Instagram 的一个已知问题有关，但通常可以通过将您的账户从个人账户重新转换为专业账户来解决。 | [将您的 Instagram 账户切换回个人资料，然后重新转换为专业账户。](https://aitoearn.helpscoutdocs.com/article/565-troubleshooting-facebook-and-instagram-connections#h_01HBEFDCSJ6GDNVG23SNCVRWP2) |
| *"看起来我们遇到了一个问题。您是否介意再试一次？"* | 如果您正在连接一个全新的 Instagram 商业账户，但尚未完成 Meta 的“设置”步骤（关注账户、添加个人资料图片、添加简介等），您会看到此错误。 | 完成 Instagram 新渠道的整个“设置”过程，然后再次尝试 Aitoearn/Instagram 的连接过程。 |
