# X/Twitter 错误库

在使用像 Aitoearn 这样的调度工具时，可能会遇到社交网络拒绝帖子并返回错误信息的情况。让我们来解决这些问题吧！在下方的“错误”列中查找您看到的错误，然后根据右侧“解决方案”列中的建议进行排查。

| **错误** | **错误信息的原因** | **解决方案** |
| --- | --- | --- |
| *"It looks like 这个帖子可能最近已经被分享过了。查看我们的常见问题解答中关于重复帖子的指南。"* | X/Twitter 检测到该帖子是重复的。[X/Twitter 对于重复发布相同或几乎相同的帖子有严格的政策](https://help.twitter.com/en/rules-and-policies/platform-manipulation)。为了检查重复性，X/Twitter 会在发布新帖子之前，将其文本与您最近的帖子进行比较。请编辑重复帖子的内容，然后重试失败的帖子。更改标题中的关键词，并确保任何重复的帖子在时间上有足够的间隔。 |
| *"The 每日 posting limit 用于这个 channel has been 触达ed! Please try reducing the amount of times per day 在您的调度和 try again later."* | 该错误表明您已达到 X/Twitter 的每日发布限制。[X/Twitter 允许您每天发布 2,400 条帖子](https://help.twitter.com/en/rules-and-policies/twitter-limits)。然而，每日限制会被分解为更小的时间段限制。X/Twitter 还限制了任何账户可以发送的私信数量以及关注的数量。请放慢发布速度。尝试在几个小时后重新发布您的帖子。 |
| *"It looks like 您可能已经重新发布了这个帖子。不幸的是，我们只能对每个帖子重新发布一次。如果该帖子是您自己的，您可以在我们的常见问题解答中找到一些重新分享该帖子的技巧。"* | X/Twitter 检测到您最近已经重新发布了该帖子。[X/Twitter 对于重复分享相同内容有严格的政策](https://help.twitter.com/en/rules-and-policies/platform-manipulation)，并会在发布新帖子之前，将其文本与您最近的帖子进行比较，以检查重复性。请不要尝试重复发布相同的内容。同时确保您调度的任何重复帖子之间有足够的时间间隔。 |
| *"Oops, it looks like 那个 X/Twitter 个人资料 was locked 何时我们 tried 到发布这个。Please visit **Twitter.com** 和 send a 帖子 from their 仪表板 到 unlock 您的账户和 then retry the 帖子."* | 该错误表明 X/Twitter 暂时锁定了您的账户，原因是检测到垃圾信息活动，我们无法代表您完成发布。 | 请访问 [X/Twitter](https://twitter.com/) 并尝试登录您的账户，以了解为什么它被暂时锁定。按照 X/Twitter 提供的任何说明操作。一旦您的账户解锁，[请刷新您在 Aitoearn 的 X/Twitter 渠道](https://aitoearn.helpscoutdocs.com/article/573-refreshing-a-channel-in-aitoearn)，然后继续发布。 |
| *"So sorry, it looks like 这个 X/Twitter 账户 has been suspended. It 可能 be worth getting 在 touch 与 X/Twitter."* | 该错误表明 X/Twitter 已暂停您账户的访问权限，我们无法代表您完成发布。 | 请访问 [X/Twitter](https://twitter.com/) 并尝试登录您的账户，以了解为什么它被暂停。按照 X/Twitter 提供的任何说明操作。一旦您的账户恢复，请[刷新您在 Aitoearn 的 X/Twitter 渠道](https://aitoearn.helpscoutdocs.com/article/573-refreshing-a-channel-in-aitoearn)，然后继续发布。有时 X/Twitter 可能会错误地暂停账户。要解决此问题，[您需要通过此处向 X/Twitter 提交申诉](https://help.twitter.com/en/forms/account-access/appeals)。 |
| *"我们无法完成此请求，因为该链接已被 X/Twitter 或我们的合作伙伴标记为可能有害。"* | 该帖子包含了一个被 X/Twitter 阻止的链接。 | 编辑帖子并移除该链接以便发布。您可以阅读更多关于 X/Twitter 阻止链接的处理方式 [这里](https://help.twitter.com/en/safety-and-security/phishing-spam-and-malware-links)，其中涵盖了他们可能阻止的链接类别（例如，可能窃取个人信息的恶意链接、误导用户或干扰用户体验的垃圾链接、仇恨行为等）。如果您认为该链接是被 X/Twitter 错误阻止的，您可以通过[此表单](https://help.twitter.com/en/forms/authenticity/spam)向他们报告。 |
| *"Sorry, direct messages 可以't currently be sent through Aitoearn."* | 该帖子以字母 D 和 @用户名开头。***D/d [用户名] + 消息*** 是 X/Twitter 的 SMS 命令之一，用于通过 SMS 向另一个 X/Twitter 个人资料发送私信。[这里是相关的 X/Twitter 指南](https://help.twitter.com/en/using-twitter/direct-messages#via-sms)。![Aitoearn composer 与 red 错误 那个 Direct Messages are not 支持ed](//d33v4339jhl8k0.cloudfront.net/docs/assets/64cb6cf86d4a3e191f237e66/images/67dc31f7c95bfc0d5610b20b/file-zMxxlvTKP6.jpg) 编辑帖子，使其不以 "D" 或 "d" 开头，然后您应该能够成功添加帖子。 |
