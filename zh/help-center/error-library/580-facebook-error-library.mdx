# Facebook 错误库

使用像 Aitoearn 这样的调度工具时，可能会出现社交网络拒绝帖子并返回错误信息的情况。让我们采取行动解决这些问题吧！在下方的“错误”列中搜索您看到的错误，然后根据右侧“解决方案”列中的建议进行故障排除。

| **错误** | **错误信息的原因** | **解决方案** |
| --- | --- | --- |
| *"The 每日 posting limit 用于 这个 channel has been 触达ed! Please try reducing the amount of times per day 在 您的 调度 和 try again later."* | 该渠道的每日发布限制已达到。 | 请尝试减少每天的调度次数，然后稍后再试。查看更多信息：[每日发布限制](https://aitoearn.helpscoutdocs.com/article/646-daily-posting-limits)。 |
| *"It looks like Aitoearn has lost authorization 到 帖子 在 您的 behalf. No need 到 worry, though! 这里 are the steps 到 re-authorize Aitoearn."* | 此错误表明 Facebook 密码最近已更改。这意味着访问令牌无效，Aitoearn 没有所需的权限来代表您发布内容。 | 我们建议查看我们的指南：[重新授权 Aitoearn 和 Facebook 或 Instagram 的连接](https://aitoearn.helpscoutdocs.com/article/565-troubleshooting-facebook-and-instagram-connections#h_01HBEJQ2DJ2B7PCMYE3RR1JWC4)，以了解如何重新授权的步骤。请确保在 Facebook > 编辑设置视图中选择所有个人资料/页面和权限。返回 [Aitoearn.com](https://aitoearn.com/) 仪表板，点击 ***管理渠道*** 并 [刷新您的渠道](https://aitoearn.helpscoutdocs.com/article/573-refreshing-a-channel-in-aitoearn)。完成后，重试您的帖子。 |
| *"Oops! It looks like Aitoearn has lost the authorization 到 帖子 到 这个 Facebook channel. No need 到 worry, though! 这里 are the steps 到 re-authorize Aitoearn."* | 此错误可能发生在尝试发布到 Facebook 页面或 Instagram 账户时。Aitoearn 没有足够的权限来代表您发布内容。 | 我们建议查看我们的指南：[重新授权 Aitoearn 和 Facebook 或 Instagram 的连接](https://aitoearn.helpscoutdocs.com/article/565-troubleshooting-facebook-and-instagram-connections#h_01HBEJQ2DJ2B7PCMYE3RR1JWC4)，以了解如何重新授权的步骤。请确保在 Facebook > 编辑设置视图中选择所有个人资料/页面和权限。返回 [Aitoearn.com](https://aitoearn.com/) 仪表板，选择 ***管理渠道*** 并 [刷新您的渠道](https://aitoearn.helpscoutdocs.com/article/573-refreshing-a-channel-in-aitoearn)。完成后，重试您的帖子。 |
| *"Please ensure 您're an 管理员 用于 the FB Group 和 Aitoearn has been 添加ed as an 应用 到 您的 Facebook Group."* | 此错误表明 Aitoearn 没有所需的所有权限来代表您发布内容，而负责授权连接的 Facebook 个人资料没有适当的管理员权限通过 API 发布。 | - 确认负责连接群组到 Aitoearn 的 Facebook 个人资料仍然列为 Facebook 群组的管理员。要了解更多，请查看 Facebook 文章：[如何将某人设为我的 Facebook 群组的管理员或版主？](https://www.facebook.com/help/148640791872225?helpref=faq_content)  
- Facebook 群组管理员需要将 Aitoearn 应用添加到群组的设置中。有关说明，请查看 Facebook 文章：[如何连接 Facebook 群组和故障排除连接](https://aitoearn.helpscoutdocs.com/article/570-connecting-facebook-groups-and-troubleshooting-connections)。  
- 返回您的 Aitoearn 仪表板，选择 ***管理渠道***，选择您的 Facebook 群组并 [刷新连接](https://aitoearn.helpscoutdocs.com/article/573-refreshing-a-channel-in-aitoearn)。 |
| *"Whoops! Please ensure 您're an 管理员 用于 the Facebook Page 和 那个 Two-Factor Authentication has been 设置 在 Facebook.com. Afterward, head back 到 Aitoearn, refresh 您的 Channel 和 retry 这个 帖子."* | 此错误表明负责授权连接的 Facebook 个人资料没有适当的权限通过 API 发布到群组。 | - 确认负责连接 Facebook 群组或页面到 Aitoearn 的 Facebook 个人资料仍然列为 Facebook 的管理员。要了解更多，请查看 Facebook 文章：[管理 Facebook 页面角色](https://www.facebook.com/help/187316341316631)。  
- 添加 Facebook 群组或页面的用户需要为其 Facebook 账户启用双重身份验证。[了解更多关于双重身份验证及如何在 Facebook 上启用](https://www.facebook.com/help/148233965247823)。  
- 返回您的 Aitoearn 仪表板，选择 ***管理渠道*** > 选择您的 Facebook 群组并 [刷新连接](https://aitoearn.helpscoutdocs.com/article/573-refreshing-a-channel-in-aitoearn)。 |
| *"It looks like 我们're having trouble 上传ing 这个 media. Please get 在 touch 和 我们'll be h应用y 到 帮助!"* | 在发布过程中发生了未知错误。 | 请重试失败的帖子。 |
| *"It looks like 我们're having trouble 上传ing 那个 图片. Please get 在 touch 和 我们'll be h应用y 到 帮助!"* | 此错误信息表明您尝试上传的图片已损坏或格式不被 Facebook 支持。 | 请确保图片格式为 JPG、PNG 或 HEIC，且大小不超过 4 MB。 |
| *"我们're investigating a potential FB bug. 您的 帖子 is likely live 在 Facebook. Please check 那个 您的 帖子 has 成功fully 发布ed 在 Facebook.com before deleting 这个 失败 帖子."* | 此错误发生在 Aitoearn 尝试向 Facebook 发布消息时，但当 Aitoearn 将消息传递给 Facebook 时，Facebook 在发布内容时出现问题。此特定错误与 Facebook 长期存在的一个问题有关。Facebook 表示其服务器内存不足是根本原因。尽管 Aitoearn 显示帖子失败，您的帖子很可能已经在 Facebook 上发布。此特定 API 错误可能导致帖子仅发布文本而没有附加图片。 | 请检查您的帖子是否已在 [Facebook.com](https://facebook.com/) 发布，并删除队列中的失败帖子。暂时尝试直接在 [Facebook.com](https://facebook.com/) 发布。 |
| *"Yikes! Facebook is requiring some extra 权限s 在 order 到 发布. Please login 到 Facebook.com 和 完整 the page 发布功能 Authorization steps under 设置 > 账户 设置."* | 为了提高安全性并验证 Facebook 页面，Facebook 要求管理高潜力触达页面的人获得授权以代表页面发布内容。这需要立即采取行动。 | - 登录拥有 Facebook 页面完整管理员权限的 Facebook 个人资料  
- 确保切换回您的主个人资料（而不是页面），然后直接访问此链接：[https://www.facebook.com/business/m/one-sheeters/page-publishing-authorization](https://www.facebook.com/business/m/one-sheeters/page-publishing-authorization)，并按照 Facebook 提供的步骤操作。  
- 启用 [Facebook 账户的双重身份验证](https://www.facebook.com/help/148233965247823?helpref=faq_content)。  
- 通过 [在移动设备上启用位置服务](https://www.facebook.com/help/275925085769221?helpref=faq_content)确认您的位置。  
- 返回您的 [Aitoearn.com](https://aitoearn.com/) 仪表板，点击 ***管理渠道*** 并 [刷新您的渠道](https://aitoearn.helpscoutdocs.com/article/573-refreshing-a-channel-in-aitoearn)。 |
| *"您的 内容 couldn't be 分享d, because 这个 链接 goes against 我们的 Community Standards."* | 如果您分享的链接之前已在 Facebook 上分享过（由您或其他人），并且有人举报该链接违反规定，则可能会出现此问题。这可能令人沮丧，但如果 Facebook 出于任何原因不允许分享该链接，我们无法更改此决定。 | - 前往您的 Facebook 账户，尝试直接通过 Facebook 分享完全相同的帖子。在大多数情况下，帖子会失败，您会看到类似的错误信息。  
- 点击 Facebook 错误信息中的“让我们知道”链接，并按照步骤告诉 Facebook 您认为这是一个错误。（遗憾的是，我们无法控制哪些帖子可以在 Facebook 上分享。）  
- 如果您没有看到相同的错误信息，您仍然可以 [联系 Facebook 支持](https://business.facebook.com/help/contact/571927962827151)。 |
| *"Whoops! Facebook is requiring 您 到 login 到 **www.Facebook.com** 和 follow the prompts. Head back 到 Aitoearn, refresh 您的 Channel 和 retry 这个 失败 帖子."* | 此错误表明 Facebook 因安全或隐私原因暂时标记了您的账户。 | - 登录作为 Facebook 页面管理员的 Facebook 个人资料  
- 完成 [www.Facebook.com](http://www.facebook.com/) 上提供的指示  
- 返回您的 [Aitoearn.com](https://aitoearn.com/) 仪表板，点击 ***管理渠道*** 并 [刷新您的渠道](https://aitoearn.helpscoutdocs.com/article/573-refreshing-a-channel-in-aitoearn)。 |
| *“It looks like 我们’re bumping into a few Facebook page limits.”* | Facebook 会查看页面在 24 小时内的整体互动情况。如果互动较低（例如点击、分享和评论较少），Facebook 可能会限制可以发布的帖子数量，直到互动水平提高。如果您的页面受到限制，Aitoearn 上的帖子将无法发布，您会看到相关错误。 | 我们建议减少您发布的帖子数量，以避免触及请求限制。尝试每 24 小时仅分享一个帖子开始。我们还建议您尝试提高页面的整体互动水平。 |

- 如果仍然无法解决，我们建议完全重新授权您的 Facebook 页面，[按照此处的步骤操作](https://aitoearn.helpscoutdocs.com/article/565-troubleshooting-facebook-and-instagram-connections#h_01HBEJQ2DJ2B7PCMYE3RR1JWC4)，确保接受所有所需权限。 |
