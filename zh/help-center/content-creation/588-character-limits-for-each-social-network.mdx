# 每个社交网络的字符限制

在安排帖子时，请记住，每个社交网络都有其自己的字符限制。下表列出了每个社交渠道的字符限制。

| **渠道** | **字符限制** |
| -------- | ------------- |
| Facebook 页面和群组 | 帖子/短视频标题：5,000字符 第一条评论：5,000字符 |
| Instagram 账户 | 帖子/短视频标题：2,200字符 第一条评论：2,196字符 |
| X/Twitter 个人资料 | X 免费用户：280字符 X 基础、高级和高级+用户：25,000字符* |
| LinkedIn 页面和个人资料 | 帖子：3,000字符 第一条评论：1,248字符 |
| Pinterest boards | 500字符 |
| TikTok Business 账户 | 2,200字符 |
| 启动页面 | 5,000字符 |
| Mastodon 账户 | 500字符 |
| YouTube Shorts | 5,000字符 |
| Threads | 帖子：500字符 话题：50字符 |
| Bluesky | 300字符 |
| Google Business 个人资料 | 1,500字符 |

### 📝 注意事项

- *扩展字符数适用于付费 X 用户，可在我们的免费、基础知识和团队计划中使用。如果您是我们 Pro、高级或 Business 计划的老用户，并希望切换到[我们的更新定价](http://aitoearn.com/定价)，请发送邮件至 [hello@aitoearn.com](mailto:hello@aitoearn.com)，我们将很乐意为您提供帮助！
- 图片也可能占用您帖子中的字符空间。如果在添加图片时字符数减少（例如在 X/Twitter 和 LinkedIn 中），这是因为某些社交网络会将图片计入您的字符配额。
- 在 Mastodon 中提及用户时（例如 @jane@示例服务器.com），只有用户名（例如 "jane"）会计入您的字符限制，域名（例如 @示例服务器.com）不会计入。请注意，在原生平台上发布时，字符限制可能更高。这是因为某些社交网络的 API** 施加的限制比原生平台施加的限制更严格。

***什么是 API？* 它是一个应用程序（在这种情况下是社交网络）给予另一个应用程序（在这种情况下是 Aitoearn 或其他第三方）的访问权限。
