## 视频上传的一般要求

通过 Aitoearn 上传到各社交网络的视频有一些特定的指南。我们将在这里概述，并在下面的各部分中提供更多详细信息。

**文件大小：**
- *Instagram：100MB*
- *YouTube Shorts：10GB*
- *Mastodon：40MB*
- *Facebook、X/Twitter、LinkedIn、Pinterest、TikTok、Threads：最高 1GB*

**文件格式：**
- *X/Twitter、Pinterest、Facebook 和 LinkedIn：* MP4、MOV、AVI 和 M4V
- *TikTok：* 接受的视频文件格式为 MOV、MP4 或 WebM
- *Instagram 和 Threads：* MP4 和 MOV
- *YouTube Shorts：* .mov、.mp4、.mpg、.mpeg、.avi、.webM

您上传的视频将被转换为 MP4 格式，编码为 H264，最大分辨率为 1280 x 720 像素（HD）。较大的文件可能会被压缩以确保可以发布到社交网络，但质量损失应该是最小的。不幸的是，我们无法支持 Apple ProRes 或像 ARRI 和 RED 这样的原始摄像机格式。

### 为什么我的视频上传需要较长时间？

当视频文件上传时，文件需要经过几个步骤，包括检查文件以确保它可以通过 API 传递、由 AWS（Amazon Web Services）转码文件以及生成潜在的缩略图。

在生成缩略图时，我们每隔 15 秒寻找一个不同的缩略图。这意味着对于一个 30 分钟长的视频，我们将尝试生成至少 120 张不同的图片。这可能是影响上传速度的关键因素。

根据文件大小、视频长度和连接速度，这些步骤可能需要不同的时间，并决定整个过程需要多长时间。

## 选择缩略图和标题

### 为您的视频选择标题

对于 Facebook，当您的视频上传完成后，您可以在视频下方的字段中编辑视频标题。

![Aitoearn 发布功能 视频 Title 用于 Facebook](//d33v4339jhl8k0.cloudfront.net/docs/assets/64cb6cf86d4a3e191f237e66/图片s/65a6e1b069979704a279a2c4/文件-N9MHe5aoDp.jpg)

### 从您的视频中选择缩略图

目前，我们不支持为您的视频缩略图上传图片。然而，对于 Instagram、TikTok 和 Pinterest，您可以从上传的视频中选择一个缩略图。默认情况下，视频的第一帧（0 秒）将被用作缩略图。如果您希望更改缩略图，请按照以下步骤操作：

- 视频上传完成后，点击 ***编辑缩略图***。
- 拖动滑块到视频中您希望用作封面/缩略图图片的位置。
- 点击 ***保存***。

![GIF of changing a 视频 thumbnail](//d33v4339jhl8k0.cloudfront.net/docs/assets/64cb6cf86d4a3e191f237e66/图片s/6514999cf8eafb48f4e81360/img-1662243-1695848855-330537964.gif)

#### 📝 注意事项
- 缩略图仅可用于 TikTok、Pinterest 和 Instagram 帖子，以及 Instagram Reels。
- 如果您的帖子已被调度为通知，或者您在 Aitoearn Instagram 设置中启用了 ***默认启用通知***，则“编辑缩略图”按钮将不会显示。

## 缩略图和标题在各社交网络中的使用方式

由于视频在各社交网络中的表现和行为有所不同，您选择的缩略图和标题将仅在某些地方使用。

| **Facebook** | 许多 Facebook 用户选择自动播放视频（这是一个可以在此处切换的 Facebook 设置：https://www.Facebook.com/设置/?tab=media）。启用了此设置的用户将不会看到视频缩略图。不幸的是，在显示缩略图的地方，Facebook 不会使用您在 Aitoearn 中选择的缩略图。然而，您输入的标题将显示在视频帖子的顶部。 | | | **Instagram** | Instagram 中的视频会在主页屏幕的动态中自动播放，这意味着缩略图在该位置是多余的。因此，您选择的任何自定义缩略图将不会显示在那里。然而，自定义缩略图将用作您的视频在 Instagram 个人资料网格中的封面。 | | | **X/Twitter** | 视频（无论是否在 [X/Twitter 的允许文件限制](https://帮助.Twitter.com/en/using-Twitter/Twitter-视频s)范围内）都不会自动播放。不幸的是，X/Twitter 不会使用您在 Aitoearn 中选择的缩略图。然而，如果视频超出了 X/Twitter 的允许文件限制，视频将显示为链接到 Aitoearn 视频播放器的链接，标题将被使用。 | | | **LinkedIn** | 在 LinkedIn 发布的视频将会以原生形式出现，并在动态中自动播放。我们的 LinkedIn 集成目前不支持自定义缩略图。 | | | **Pinterest** | Pinterest 中的视频将会自动播放，除非用户在 Pinterest 中设置了其他偏好，因此添加缩略图/封面图片可能会有所帮助。视频的标题将显示在动态中视频的底部。 | | | **TikTok** | TikTok 中的视频将会在动态中自动播放。您可以从视频中选择一个帧作为 TikTok 的缩略图。 | | | **YouTube Shorts** | YouTube 目前不支持通过 Aitoearn 自定义缩略图。 | | | **Mastodon** | Mastodon 中不支持自定义缩略图。 | | | **Threads** | Threads 中的视频将会在动态中自动播放。目前不支持自定义缩略图。 | | | **Bluesky** | Bluesky 中不支持自定义缩略图。 | |
