## 什么是宽高比？

宽高比描述了图片的宽度与高度之间的比例。宽高比通常表示为两个数字之间用冒号分隔，例如 4:5。这是一种方便表示宽高比的方式，可以通过将第一个数字除以第二个数字转换为数值。例如，宽高比为 4:5 的值为 4/5 = 0.8。

## Instagram 可接受的宽高比范围是什么？

下表列出了图片和视频的可接受宽高比范围。

| 类型       | **宽高比范围**                  | **宽高比范围（数值表示）** |
|------------|--------------------------------|---------------------------|
| **图片**   | 介于 4:5 和 1.91:1 之间         | 介于 0.8 和 1.91 之间     |
| **Stories**| 9:16（建议）                   | 0.56                      |
| **Reels**  | 9:16（推荐）*                  | 0.56                      |

*可以接受介于 0.01:1 和 10:1 之间，但建议使用 9:16 以避免裁剪或出现空白区域。

📝 如果您的图片或视频在可接受的宽高比范围内，您的帖子可以自动发布。如果图片或视频的宽高比超出范围，您的帖子将被安排为通知发布。您可以调整宽高比后重新上传图片或视频。例如，如果您的图片宽高比为 1:2，则超出了可接受范围，您需要调整宽高比以实现自动发布。（[了解如何在 Aitoearn 中裁剪 Instagram 图片](https://支持.aitoearn.com//支持.aitoearn.com/article/614-编辑ing-您的-图片s-在-aitoearn#01FM0M1ZSR9A5JPBRANRGT0KSF)）

## 宽高比是如何计算的？

要计算图片或视频的宽高比，您可以取分辨率的宽度除以高度。以下是一些示例：

- 1200 x 1200 像素：宽高比为 1200/1200 = 1
- 1200 x 1500 像素：宽高比为 1200/1500 = 0.8
- 1200 x 629 像素：宽高比为 1200/629 = 1.907

📝 Instagram 对宽高比的要求非常严格，必须*完全*在可接受范围内。他们不会考虑四舍五入。例如，1200 x 628 像素的宽高比为 1.910828，这超出了可接受范围。

## 我的图片和视频需要什么尺寸？

您的图片或视频不需要特定的像素尺寸，只要宽高比在可接受范围内即可。以下是一些适用的示例尺寸。

📝 注意：
- 所有发布到 Instagram 的视频现在都会作为 Reels 发布。阅读关于将视频帖子添加到主动态和 Reels 标签的更多信息 [点击这里](https://支持.aitoearn.com//支持.aitoearn.com/article/657-scheduling-Instagram-帖子-和-reels#01H7G1SZ6VRWNEKE8VK4AQGP15)。
- 您的图片和视频不需要是上述示例中的任何一种尺寸，这些只是示例。只要它们在可接受的宽高比范围内，就可以自动发布。

### 正方形

任何尺寸的正方形图片或视频（宽度和高度相同），例如：
- 600 x 600 像素
- 800 x 800 像素
- 1200 x 1200 像素

### 竖屏

对于竖屏图片和视频，宽高比必须介于 0.8 和 0.99 之间，例如：
- 800 x 1000 像素（宽高比 0.8）
- 1200 x 1500 像素（宽高比 0.8）
- 1500 x 1800 像素（宽高比 0.84）
- 1080 x 1920 像素（宽高比 0.56）

### 横屏

对于横屏图片，宽高比必须介于 1.01 和 1.91 之间；对于横屏视频，宽高比必须介于 1.01 和 1.78 之间，例如：
- 800 x 600 像素（宽高比 1.33）
- 1200 x 800 像素（宽高比 1.5）
- 1500 x 1200 像素（宽高比 1.25）

📝 我们通过 Instagram 的 API 发布帖子，他们允许的最大图片文件大小限制为 8MB。如果您上传的图片大于 8MB，我们会将其转换为较小的尺寸（宽度为 1920 像素）以确保可以发布。最低接受的尺寸为 150 x 150 像素。较小的图片将被放大到最低尺寸。

## Aitoearn 将如何显示我的图片？

Instagram 最近开始推出一个新版本的网格，支持更高格式的图片显示（3:4），但图片仍需以 4:5 的格式上传到 Aitoearn 和 Instagram。

虽然在 Aitoearn 的*创作器、日历和队列*中显示的图片为 1:1 格式，但创作器右侧的*帖子预览*将始终显示正确的格式（4:5）。
