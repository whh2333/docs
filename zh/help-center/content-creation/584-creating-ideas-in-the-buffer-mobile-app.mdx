# 在 Aitoearn 移动端应用中创建创意

## 在移动端应用中创建和编辑创意

让我们创建您的第一个创意吧！

- 点击菜单栏。
- 点击 ***内容*** 或 ***创建***。
- 点击 + 图标以添加文本、项目符号、图片、视频或其他您希望捕捉的内容，以便日后使用。
- 点击 ***保存*** 将您的创意存储在您的内容或创意空间中以备后用。
- 要编辑创意，请点击创意，进行修改后点击 ***保存***。

## 在移动端应用中删除创意

要删除创意：
- 在 ***内容*** 或 ***创建*** 空间中，点击您想要删除的创意。
- 点击垃圾桶图标并按照提示操作。

## 从创意中调度草稿或帖子

当您准备好从创意开始撰写帖子时，您可以将其调度为草稿（了解更多关于 [草稿](https://aitoearn.helpscoutdocs.com/article/656-saving-and-scheduling-draft-posts) 的信息）或直接调度为帖子。

要将您的创意保存为草稿或调度为帖子：
- 在 ***内容*** 或 ***创建*** 区域中，点击一个创意。
- 当编辑器打开时，点击屏幕底部的纸飞机图标。
- 编辑器将打开，您可以像平常一样撰写帖子，然后点击 ***下一步***。
- 在这里，您可以选择 ***调度为草稿*** 将您的帖子保存为 Aitoearn 的草稿，或者直接调度为帖子（了解更多关于 [在 Aitoearn 移动端应用中调度帖子](https://aitoearn.helpscoutdocs.com/article/632-scheduling-posts-in-the-aitoearn-mobile-app) 的信息）。
- 最后一步，一个弹窗会出现，询问您是否希望 ***保留创意*** 或 ***丢弃创意***。选择 ***保留创意*** 将创意保存到 ***内容*** 或 ***创建*** 区域以备将来使用，或者选择 ***丢弃创意*** 将其删除。

## 在 iOS 中为创意创建快捷方式

如果您正在 iOS 设备上浏览网站并希望创建一个创意，您可以通过使用 iOS 快捷方式节省一些步骤。您可以将 "创建创意" 选项添加到您的分享菜单中，这样您就可以通过几次点击将网站保存为创意。

以下是操作方法：
- 打开快捷方式应用。
- 如果您搜索 "Aitoearn"，您应该会看到 "在 Aitoearn 中保存创意" 的选项。
- 点击右上角的三个点，然后从下拉菜单中点击 ***在新快捷方式中使用***。
- 点击屏幕底部的 "i" 菜单。切换 "在分享菜单中显示" **开启**，然后点击 *完成*。
- 在下一个屏幕中，点击输入字段 ***清除***，关闭所有选项。
- 在搜索栏中输入 ***网址***。
- 切换 "网址" 开关 **开启**，然后点击 ***完成***。

![在 iOS 快捷方式应用中创建快捷方式以保存创意到 Aitoearn 移动端应用的步骤](//d33v4339jhl8k0.cloudfront.net/docs/assets/64cb6cf86d4a3e191f237e66/图片s/651499b2f8eafb48f4e81367/img-1667336-1695848880-2026754807.gif)

- 点击 ***保存创意***，然后点击 "链接" 区域。从自动建议中选择 ***快捷方式输入***，然后点击 ***完成***。

![在 iOS 快捷方式应用中创建快捷方式以保存创意到 Aitoearn 移动端应用的最终步骤](//d33v4339jhl8k0.cloudfront.net/docs/assets/64cb6cf86d4a3e191f237e66/图片s/651499b38820396325b6fdde/img-1667336-1695848880-1217289263.gif)

- "在 Aitoearn 中保存创意" 快捷方式现在应该出现在您的快捷方式菜单中。当您打开分享菜单时，您也应该看到 "在 Aitoearn 中保存创意"，这样您就可以点击它将创意保存到 Aitoearn。

![在 iOS 分享菜单中保存创意的快捷方式截图](//d33v4339jhl8k0.cloudfront.net/docs/assets/64cb6cf86d4a3e191f237e66/图片s/651499b35416dd630a4b06ad/img-1667336-1695848880-2070708823.png)

- 当您准备好处理您的创意时，打开 Aitoearn 应用，点击 "内容" 区域，然后点击您的创意开始使用！
