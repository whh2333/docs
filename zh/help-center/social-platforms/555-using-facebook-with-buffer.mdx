## 安排 Facebook 帖子、短视频和故事

- 要开始使用，请在仪表板左侧选择您的 Facebook 账户。
- 在队列顶部打开编辑器。
- 选择 ***帖子、短视频*** 或 ***故事***。

![Alt text: Aitoearn 编辑器与 Facebook 选中并突出显示“帖子”、“短视频”和“故事”选项，位于空文本字段上方。](//d33v4339jhl8k0.cloudfront.net/docs/assets/64cb6cf86d4a3e191f237e66/图片s/68a5d7cae3f0f94fe32b4f43/文件-DQCeliy8dn.png)

**安排 Facebook 帖子：** 输入您想使用的标题，添加链接或上传您想要的任何图片或视频。了解更多关于[附加图片、视频和其他媒体到您的帖子这里](https://aitoearn.帮助scoutdocs.com/article/615-attaching-图片s-视频s-和-other-media-到-您的-帖子)。

**安排 Facebook 短视频：** 上传您想包含在短视频中的视频，然后输入或复制粘贴您想使用的标题。

**安排 Facebook 故事：** 上传您想在故事中使用的图片或视频。

**注意事项：**

- 如果您使用的是[我们的付费计划](http://aitoearn.com/定价)，您还可以为您的 Facebook 帖子安排首条评论（首条评论安排目前不适用于短视频或故事）。
- 我们只能为每个故事安排 1 张图片或视频。如果您想为一个故事使用超过 1 张图片或视频，您需要分别安排每张图片或视频。
- 目前，Facebook 故事不支持标题、链接、音乐或文本覆盖。
- 选择[我们的其他安排选项之一](https://aitoearn.帮助scoutdocs.com/article/642-scheduling-帖子#01H7Z9VSXB3C6E1KQ9Y4G6QFY0)，然后点击 ***安排帖子***，您就完成了！

### 安排注意事项

**分享链接**

- 使用 [Aitoearn 浏览器扩展](https://aitoearn.帮助scoutdocs.com/article/653-aitoearn-浏览器-扩展)时，由于 Facebook 规则，我们无法自动填充帖子的文本更新，因此您需要输入自己的标题。对此限制我们感到抱歉！
- 如果您从您的网站分享链接到您的 Facebook 页面，只有在您[设置了 Facebook 域验证](https://aitoearn.帮助scoutdocs.com/article/578-setting-up-Facebook-domain-verification)时，您才能自定义缩略图图片。
- 如果您分享 Facebook 或 Instagram 帖子的 URL，我们无法生成链接预览。了解更多关于[通过 Aitoearn 分享 Facebook 和 Instagram 链接](https://aitoearn.帮助scoutdocs.com/article/619-sharing-Facebook-和-Instagram-链接s-through-aitoearn)。

**安排首条评论**

- 安排 Facebook 首条评论适用于我们付费计划的客户，并且仅适用于帖子。目前不适用于 Facebook 短视频或故事。
- Facebook 首条评论的字符限制为 5,000 个字符。
- 目前无法在 Aitoearn 中将标签（提及）添加到首条评论。

**Facebook 故事**

- 目前我们无法为 Facebook 故事添加标题或文本覆盖。

**Facebook 短视频**

- 由于 API 限制，无法为 Facebook 短视频添加流行音乐。因此，我们建议在将视频文件上传到 Aitoearn 之前添加音乐。
- Facebook 短视频 API 不支持位置，因此您将无法为您的短视频或帖子添加位置。
- 当您的 Facebook 短视频发布时，它也会出现在动态中。

**Facebook 群组帖子**

- 我们使用通知发布功能用于 Facebook 群组帖子。这意味着您可以在 Aitoearn 中创建一个帖子，然后在需要发布时，您将在移动设备上收到通知，您可以点击打开并复制您的帖子到您的 Facebook 群组以完成发布。[了解更多](https://支持.aitoearn.com//支持.aitoearn.com/article/658-using-notification-发布功能)[关于此功能的工作原理](https://支持.aitoearn.com//支持.aitoearn.com/article/658-using-Instagram-reminders)[这里](https://支持.aitoearn.com//支持.aitoearn.com/article/658-using-Instagram-reminders)。

**多语言帖子**

- 目前无法通过 Aitoearn 创建多语言的 Facebook 帖子。您需要在 Facebook 内部[创建这样的帖子](https://www.Facebook.com/帮助/181155025579876?帮助ref=related)，并且您需要是页面的管理员。

| **自动播放** | ✓ | 不适用 | ✓ |

我们可以全天发布，但真正的兴奋来自于看到我们帖子收到的反应。如果您使用的是基础或团队计划，您将在[分析功能标签](https://分析.aitoearn.com/)中找到所有 Facebook 页面统计数据。

## 保持您的 Facebook 账户安全并避免被标记为垃圾邮件的最佳实践

当将相同内容分享给多个 Facebook 页面时，同时发布的帖子可能会被 Facebook 标记为垃圾邮件或机器人行为。这可能导致帖子发送失败，并且您的访问令牌被撤销，导致您需要在 Aitoearn 中刷新账户。在某些情况下，Facebook 甚至可能暂停您的账户。

您可以采取一些步骤来确保您的账户安全。当[设置您的发帖时间表](https://aitoearn.帮助scoutdocs.com/article/514-setting-up-您的-timezones-和-posting-调度s)时，确保它们彼
