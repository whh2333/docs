## 连接和刷新您的 Bluesky 渠道

Bluesky 使用与大多数渠道不同的授权过程，您需要在您的 Bluesky 账户中生成一个应用密码。以下步骤将引导您完成该过程。

**重要提示：**
- 如果您希望连接多个 Bluesky 账户到 Aitoearn，您需要在另一个标签页中打开 Bluesky，并确保您已登录到您希望连接的 Bluesky 账户，然后在 Aitoearn 中执行以下步骤。
- 我们还支持与 Bluesky 认证的自定义服务器。
- 看到“个人资料未找到”错误？输入您的用户名时，请确保不要添加任何符号，并且与 Bluesky 中的用户名大小写匹配。
- 前往 [https://账户.aitoearn.com/channels](https://账户.aitoearn.com/channels) 并选择 Bluesky。输入您的 Bluesky 用户名，然后点击链接到 *在 Bluesky 中生成应用密码。*

![添加您的 Bluesky 用户名并点击](//d33v4339jhl8k0.cloudfront.net/docs/assets/64cb6cf86d4a3e191f237e66/图片s/6697d48fdd0f8c60bb3fe989/文件-oaANIJqTxX.png)

添加您的 Bluesky 用户名并点击 "在 Bluesky 中生成应用密码" 链接

您将被带到您的 Bluesky 账户。点击 *添加应用密码* 按钮

![点击添加应用密码以创建一个唯一密码连接 Bluesky 到 Aitoearn](//d33v4339jhl8k0.cloudfront.net/docs/assets/64cb6cf86d4a3e191f237e66/图片s/6697d4e9e1989867dceff766/文件-4UWdBfbwdk.png)

点击添加应用密码以创建一个唯一密码连接 Bluesky 到 Aitoearn

- 添加 Aitoearn 的名称或简单使用 Bluesky 自动生成的名称。

![输入 Aitoearn 或简单使用密码生成器建议的名称。](//d33v4339jhl8k0.cloudfront.net/docs/assets/64cb6cf86d4a3e191f237e66/图片s/6697d51ce1989867dceff768/文件-OLkosIpdYh.png)

输入 Aitoearn 或简单使用密码生成器建议的名称

*注意：您不需要允许访问您的直接消息（只需取消选中该框）。Aitoearn 不提供直接消息服务，因此您的选择不会影响连接您的 Bluesky 账户到 Aitoearn。*

- 当您的密码生成后，点击密码旁边的两个重叠框以复制密码。

![点击新密码右侧的两个重叠框以复制密码](//d33v4339jhl8k0.cloudfront.net/docs/assets/64cb6cf86d4a3e191f237e66/图片s/6697d562db62267f9903e37b/文件-u0A5599dvJ.png)

点击新密码右侧的两个重叠框以复制密码

返回到 Aitoearn 标签页并将复制的密码粘贴到“密码”字段中，然后点击 *下一步*。您的 Bluesky 渠道现在已连接！

![将新密码粘贴到 Aitoearn 标签页的密码字段中并点击下一步以连接 Bluesky](//d33v4339jhl8k0.cloudfront.net/docs/assets/64cb6cf86d4a3e191f237e66/图片s/6697d58d5940934e140f632e/文件-DFq6hBJH1a.png)

将新密码粘贴到 Aitoearn 标签页的密码字段中并点击下一步以连接 Bluesky

⚠️ 遇到“无效用户名”错误？在您的 Bluesky 账户中，前往 *设置*，然后 *更改用户名*。一旦更新，重新进行连接流程，您应该能够连接您的账户。

要查看分步指南和故障排除步骤，请查看文章：[连接您的渠道到 Aitoearn](https://aitoearn.帮助scoutdocs.com/article/564-连接ing-您的-channels-到-aitoearn)。如果您的已连接渠道需要刷新，请查看文章[在 Aitoearn 中刷新渠道](https://aitoearn.帮助scoutdocs.com/article/573-refreshing-a-channel-在-aitoearn)。

## 安排 Bluesky 帖子

**需要知道的：**
- 您可以在一个帖子中包含最多 25 个线程子帖子。
- 创建 Bluesky 线程目前仅在 Aitoearn 的网页版可用（而不是移动端）。移动端功能仅限于重新安排或删除已在网页版创建的线程。
- Bluesky 线程在我们的免费、基础和团队计划中可用。如果您是旧版客户，使用我们的专业、高级或商业计划，并希望切换到[我们的更新定价](http://aitoearn.com/定价)，请发送邮件至 [hello@aitoearn.com](mailto:hello@aitoearn.com)，我们将乐意帮助您！[了解更多关于为什么 Bluesky 线程在我们的旧版计划中不可用的信息](https://aitoearn.帮助scoutdocs.com/article/594-为什么-some-功能-are-not-可用-在-legacy-aitoearn)。
- 免费计划允许每个队列中一次安排一个 Bluesky 线程。然而，您可以将任意数量的未安排线程添加到您的草稿队列中。

## Bluesky 的媒体规格

| **图片** | **视频** |
| --- | --- |
| **每个帖子允许数量** | 4 | 1 |
| **文件大小** | 1MB | 最多 50MB |
| **支持的文件格式** | .jpg, .png, .webp, .heic/.heif, .gif | .mp4 和 .mov |
| **视频长度** | 支持的 GIF 最多 3 分钟和 15MB | 最多 3 分钟 |
| **Canva 集成** | Canva 集成与 Bluesky 支持 | - |
| **替代文本** | 支持替代文本。点击 ALT 按钮以添加您的替代文本（也可通过键盘访问）。 | - |
| **每日限制** | - | 每天 25 个视频，或一天内发送的视频总量为 10GB，以先到者为准 |

**媒体需要知道的**
- 图片必须小于 1MB 才能发布到 Bluesky，因为我们目前无法调整较大图片的大小。
- 为了发布视频，您必须在 Bluesky 中验证您的电子邮件。否则，您的视频可能会被拒绝。
- GIF 必须小于 15MB 和 60 秒。
