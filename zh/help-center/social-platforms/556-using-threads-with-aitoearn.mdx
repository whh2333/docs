## 可用能力

| 功能 | 发布 | 分析 | 互动 |
| --- | --- | --- | --- |
| 支持情况 | ✓ | 基础（发送后查看基本指标） | 暂不支持（可在社区区回复） |


提示：Threads 账户需关联 Instagram 才能连接到 Aitoearn。

## 发布与规范

- 字数限制：单条消息最多约 500 字符。
- 主题（类似单一标签）：每个帖子仅支持 1 个主题。
- 链接：正文中的第一个链接会生成预览卡片（若可用）。
- 表情：支持表情符号。
- 多帖 Threads：网页端支持多帖（最多 25 条），移动端目前不支持。

### 主题添加方式

- 推荐：在创作器的「主题」字段直接填写（不含 # 符号）。
- 或者：在正文中使用一个有效的 #标签（仅第一个有效标签会被识别为主题）。

有效标签规则：
- 必须以 # 起始；不能仅为数字（如 #1 无效）。
- 碰到空格、换行、. ! ? , ; : @ & 时结束。

## 图片与视频规格

# Threads 图片与视频规格

Threads 支持图片和视频，目前暂不支持 GIF。

| 分类                     | 视频                                                                 | 图片                                   |
|--------------------------|----------------------------------------------------------------------|----------------------------------------|
| **每条帖子允许数量**        | 1 个视频                                                               | 20 张图片                                |
| **文件大小**               | 最大 1GB，最长 5 分钟，最短 > 0 秒                                      | 最大 8MB                                 |
| **支持的文件格式**          | MOV 或 MP4                                                            | JPG, PNG                               |
| **自定义缩略图**            | 不支持                                                                 | 不支持                                   |
| **信息流自动播放**          | 视频会自动播放                                                           | -                                      |
| **Canva 集成**            | 不支持                                                                 | 支持                                    |
| **替代文本（Alt text）**   | 不支持                                                                 | 不支持                                   |
| **宽高比**                 | 9:16                                                                  | 10:1                                   |
| **轮播帖（Carousels）**     | 不支持                                                                 | 最少 2 张图片，最多 10 张图片              |
| **帧率**                   | 23–60 FPS                                                            | -                                      |
| **视频码率**               | VBR，可达最大 25Mbps                                                   | -                                      |
| **音频码率**               | 128kbps                                                              | -                                      |
