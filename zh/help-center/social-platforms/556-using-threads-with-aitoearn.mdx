# 使用 Threads 与 Aitoearn

Meta Threads 现已支持连接到 Aitoearn。完成连接后，您可以在桌面端编写与排期 Threads 帖子。

## 可用能力

| 功能 | 发布 | 分析 | 互动 |
| --- | --- | --- | --- |
| 支持情况 | ✓ | 基础（发送后查看基本指标） | 暂不支持（可在社区区回复） |

## 连接与刷新

1. 登录 Aitoearn，进入「频道管理」。
2. 点击「添加频道」并选择「Threads」。
3. 使用与 Threads 关联的 Instagram 账户完成授权。
4. 若授权超过 90 天失效，请在「频道管理」中执行刷新。

提示：Threads 账户需关联 Instagram 才能连接到 Aitoearn。

## 发布与规范

- 字数限制：单条消息最多约 500 字符。
- 主题（类似单一标签）：每个帖子仅支持 1 个主题。
- 链接：正文中的第一个链接会生成预览卡片（若可用）。
- 表情：支持表情符号。
- 多帖 Threads：网页端支持多帖（最多 25 条），移动端目前不支持。

### 主题添加方式

- 推荐：在创作器的「主题」字段直接填写（不含 # 符号）。
- 或者：在正文中使用一个有效的 #标签（仅第一个有效标签会被识别为主题）。

有效标签规则：
- 必须以 # 起始；不能仅为数字（如 #1 无效）。
- 碰到空格、换行、. ! ? , ; : @ & 时结束。

## 图片与视频规格

| 项目 | 视频 | 图片 |
| --- | --- | --- |
| 数量 | 每帖 1 个视频 | 每帖最多 20 张 |
| 时长/大小 | ≤ 5 分钟，≤ 1GB | 每张 ≤ 8MB |
| 格式 | MP4 / MOV | JPG / PNG |
| 比例 | 9:16（推荐） | 最宽 10:1（建议竖图/方图） |
| 帧率/码率 | 23–60 FPS，VBR ≤ 25Mbps | - |
| 替代文本 | 暂不支持 | 暂不支持 |

## 基础指标查看

发送后可在「发布」→「已发送」中查看基本指标（如点赞、回复、浏览等）。Threads 暂未开放完整分析接口。

## 常见问题（FAQ）

**Q：为什么没有看到连接 Threads 的选项？**
- 需要使用我们的新套餐；Threads 账户需为公开状态，并且已关联 Instagram。

**Q：为何主题未生效？**
- 确保每个帖子只使用 1 个主题；正文仅第一个有效 #标签会被识别；或改用「主题」字段填写。

**Q：报错 “Threads API feature not available” 怎么办？**
- 该错误通常表示当前账户未满足 Threads API 使用条件。请确认 Threads 已关联 Instagram 并为公开状态，随后重新授权。
