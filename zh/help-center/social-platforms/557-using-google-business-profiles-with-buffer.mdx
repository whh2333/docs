## 连接和刷新您的 Google 商家个人资料

#### 📝 连接清单
- Google 商家个人资料是否已验证？*必须验证！*
- 您是否登录了与您的 Google 商家个人资料关联的 Google 账户？
- Google 商家个人资料管理员是否创建了“群组”？*如果 Google 商家个人资料管理员中创建了“群组”，我们目前无法连接。*
- *解决方法：请邀请一个未创建“群组”的新 Google 账户来管理 Google 商家个人资料，然后尝试使用该管理员账户进行连接。*

您可以将多个 Google 商家个人资料连接到 Aitoearn。要将 Google 商家个人资料连接到您的 Aitoearn 账户，您需要确保：
- 您已登录与您的 Google 商家个人资料关联的 Google 账户
- 您的个人资料必须经过验证

要查看分步指南和故障排除步骤，请查看文章：[连接您的渠道到 Aitoearn](https://aitoearn.帮助scoutdocs.com/article/564-连接ing-您的-channels-到-aitoearn)。

如果您已连接的 Google 商家个人资料需要刷新，请查看文章：[在 Aitoearn 中刷新渠道](https://aitoearn.帮助scoutdocs.com/article/573-refreshing-a-channel-在-aitoearn)。

## Google 商家个人资料的图片规格

Google 商家个人资料支持图片。由于 API 限制，Google 商家个人资料不支持 GIF 或视频。

| **图片** | |
| **每个帖子允许的数量** | 1 |
| **文件大小** | 10Kb 到 10MB |
| **支持的文件格式** | JPG、PNG、WebP（上传为 WebP 的图片将转换为 PNG）、HEIC |
| **推荐分辨率** | 720px x 720px |
| **最大分辨率** | 10,000px x 10,000px |
| **最小分辨率** | 250px x 250px |
| **Canva 集成** | 支持 |
| **Alt 文本** | 支持。了解更多关于[为图片添加 Alt 文本](https://aitoearn.帮助scoutdocs.com/article/618-添加ing-alt-text-到-您的-图片s)。 |

## 常见问题

### 我的活动和优惠帖子的时区是什么？

活动和优惠的时间将使用您的 Google 商家个人资料位置的本地时区。例如，如果您创建了一个活动类型的帖子，并且活动时间是两周后的晚上 7 点，那么时间将是您 Google 商家个人资料位置的本地时间。
