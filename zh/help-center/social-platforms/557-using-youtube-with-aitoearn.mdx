# 在 Aitoearn 中使用 YouTube

YouTube 是全球最大的视频平台，适合品牌传播、知识分享与商业变现。Aitoearn 帮助您进行内容规划、批量发布与数据分析。

## 🔗 连接 YouTube 频道
1. 登录 Aitoearn 账户
2. 前往「频道管理」
3. 点击「添加频道」并选择「YouTube」
4. 使用 Google 账户完成授权

## 🎬 发布与规范
- 支持长视频与短视频（Shorts）
- 建议分辨率：1920x1080（16:9）或 1080x1920（9:16）
- 支持字幕、缩略图自定义与标签

发布步骤：上传视频 → 编辑标题/描述/标签 → 设置缩略图 → 设定发布时间（含 Shorts）

## 📊 核心指标
- 展示次数、点击率（CTR）
- 观看时长、平均观看百分比
- 订阅者变化、受众地域与年龄段

## 💡 增长建议
- 缩略图对比测试与标题优化
- 保持节奏与系列化内容
- 合理使用章节（Timestamps）与卡片/片尾屏幕
