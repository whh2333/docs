## YouTube Shorts 如何与 Aitoearn 配合工作

在使用 YouTube 时，有两种发布方式：

***自动发布功能*** - 在 Aitoearn 中调度您的 Shorts，我们会自动为您发布。

***通知发布功能*** - 在 Aitoearn 中调度您的 Shorts，您会在手机上收到通知，提醒您发布 Shorts，您可以在 YouTube 应用中完成发布。

您可以在编辑器中看到这些选项，选择 **自动发布**（如果您的帖子可以自动调度）或 **通知我**（如果您选择接收移动端通知或帖子包含无法自动发布的内容，例如超过时长限制的视频）。

![从自动发布切换到通知我功能](//d33v4339jhl8k0.cloudfront.net/docs/assets/64cb6cf86d4a3e191f237e66/图片s/66df610e811a2434cb6d3550/文件-K3QO0Ahlnl.gif)

⚠️ 如果您已连接 YouTube 账户并希望使用基于通知的发布功能，您必须在您的移动设备上 [设置通知](https://aitoearn.帮助scoutdocs.com/article/658-using-Instagram-reminders)，以确保不会错过任何帖子。（在 [iOS](https://应用s.应用le.com/us/应用/aitoearn-计划-调度-帖子/id490474324) 或 [Android](https://play.google.com/store/应用s/details?id=org.aitoearn.android&hl=en_US&gl=US&pli=1) 上下载我们的移动应用）。

## 连接和刷新您的 YouTube 渠道

前往您的渠道区域以连接您的 YouTube 渠道：[https://账户.aitoearn.com/channels](https://账户.aitoearn.com/channels)

您可以将多个 YouTube 渠道连接到 Aitoearn。为了确保 Aitoearn 知道您想连接哪个渠道，请在尝试连接之前，在另一个浏览器标签页中登录拥有该 YouTube 渠道的 Google 账户。

查看分步指南和故障排除步骤，请参阅文章：[连接您的渠道到 Aitoearn](https://aitoearn.帮助scoutdocs.com/article/564-连接ing-您的-channels-到-aitoearn)。

如果您的已连接 YouTube/X 渠道需要刷新，请参阅文章：[在 Aitoearn 中刷新渠道](https://aitoearn.帮助scoutdocs.com/article/573-refreshing-a-channel-在-aitoearn)。

### 📝 注意事项：
- 由于 API 限制，您必须以 **所有者** 身份登录 Google 才能将 YouTube 渠道连接到 Aitoearn。不幸的是，如果您是品牌账户的管理员，我们无法连接到该 YouTube 渠道。
- 为了访问 YouTube 的全部功能，请确保 [验证您的电子邮件](https://my账户.google.com/identity-document/submit?pli=1)。
- 未验证的用户在 24 小时内只能发布 10 个帖子。他们也无法在帖子描述中添加外部链接。验证用户则没有发布限制。

## 调度 YouTube Shorts

查看下表以了解调度 YouTube Shorts 帖子时可用的功能。

### 限制
- 分析中的“最佳发布时间”功能不可用于 YouTube，但有关创建最佳发布时间表的建议，请查看我们的博客文章：[在 YouTube 上发布的最佳时间](https://aitoearn.com/资源/best-time-到-帖子-在-您Tube/)。
- YouTube 社区发布功能无法通过 Aitoearn 使用。如果您希望看到此功能，请随时在我们的 [建议板](https://suggestions.aitoearn.com/p/您Tube-帖子-到-community-tab) 上投票。
- Aitoearn 自动发布的 Shorts 无法使用 YouTube 的音乐库中的声音和音乐。如果您希望为 Shorts 添加声音，请使用我们的 [通知发布功能](https://支持.aitoearn.com//支持.aitoearn.com/article/658-using-notification-发布功能)进行调度。

| **功能 & 注意事项** | **说明** |
| **调度帖子** | 查看我们的分步指南以 [调度帖子](https://aitoearn.帮助scoutdocs.com/article/642-scheduling-帖子) 到任何社交渠道，或浏览以下 YouTube 特定细节。 |
| **每日发布限制** | 未验证电子邮件地址的用户每 24 小时最多发布 10 个帖子。验证电子邮件地址的用户每 24 小时无限制。了解更多关于 [每日发布限制](https://aitoearn.帮助scoutdocs.com/article/646-每日-posting-limits)。 |

| **标签** | 在编辑器中输入您想使用的标签。帖子发布后，标签将会像在原生平台中一样被超链接。了解更多关于 [标签如何在 Aitoearn 中工作](https://aitoearn.帮助scoutdocs.com/article/586-how-hashtags-work-在-aitoearn)。 |
| **缩略图** | 请注意，我们目前不支持为 YouTube Shorts 选择自定义缩略图。 |

## YouTube Shorts 的视频规格

YouTube 仅支持视频，且无法创建没有视频的 YouTube 帖子。Aitoearn 目前仅支持 ***YouTube Shorts***，暂不支持长视频。

| **视频** | **说明** |
| **每个帖子允许的视频数量** | 1 |
| **视频长度** | 最长 3 分钟 |
| **文件大小** | 10GB 或以下 |
| **尺寸 / 纵横比** | 方形 (1:1) 或竖屏 (9:16)。推荐使用竖屏，在移动设备上效果最佳。 |
| **支持的文件格式** | .mov, .mp4, .mpg, .mpeg, .avi, .webM |
| **支持自定义缩略图** | 否 |
| **Giphy 和 Canva 集成** | 否 |
| **Alt 文本** | 无法通过 Aitoearn 为 YouTube 视频添加 Alt 文本。了解更多关于 [为图片添加 Alt 文本](https://aitoearn.帮助scoutdocs.com/article/618-添加ing-alt-text-到-您的-图片s)。 |

我们可以全天发布帖子，但真正的乐趣在于看到我们的帖子反响如何。您可以通过进入 ***发布*** 标签，然后进入 ***已发送*** 标签，查看每个 YouTube Shorts 的分析功能（观看次数、点赞和评论）。

## 常见问题

### 是否可以更改 YouTube Shorts 的默认类别？

目前，Aitoearn 不支持更改发布时 YouTube Shorts 的默认类别。YouTube 会自动为 Shorts 分配类别，此设置无法通过 Aitoearn 自定义。如果您希望在发布后更改 Shorts 的类别，您需要直接在 YouTube Studio 中更新。
