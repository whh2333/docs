## 连接和刷新您的 Threads 个人资料

您可以连接多个 Threads 个人资料到 Aitoearn。然而，您的 Threads 账户需要连接到一个 Instagram 账户才能连接到 Aitoearn。要查看逐步指南和故障排除步骤以连接您的 Threads 个人资料，请查看文章：[连接您的渠道到 Aitoearn](https://aitoearn.帮助scoutdocs.com/article/564-连接ing-您的-channels-到-aitoearn)。如果您的已连接 Threads 渠道需要刷新，请查看文章[在 Aitoearn 中刷新渠道](https://aitoearn.帮助scoutdocs.com/article/573-refreshing-a-channel-在-aitoearn)。

📝 **重要提示：**
- 您的 Threads 账户必须连接到一个 Instagram 账户才能连接到 Aitoearn。
- 如果您的 Threads 渠道设置为私密，我们使用的访问令牌将在 90 天后过期，需要您刷新您的渠道。

**连接多个 Threads 个人资料 -** Aitoearn 将自动尝试连接到您在浏览器中登录的账户，因此我们建议先登录 [threads.net](http://threads.net) 使用您想要连接到 Aitoearn 的账户，然后前往 Aitoearn 连接渠道。

## 添加主题到 Threads 帖子

在 Aitoearn 中创建 Threads 帖子时，您可以附加一个**主题**以帮助组织您的内容并提高可发现性。主题充当内容标签，将相关帖子在 Threads 中分组在一起，使关注者更容易探索特定主题或对话。您可以在 Aitoearn 中调度 Threads 帖子时通过两种方式添加主题。

### 选项 1：在“主题”字段中添加主题（推荐）

为了确保您的主题正确应用，我们建议使用创作器中的**主题**字段。它为您提供完全控制，并确保您的已调度内容的一致性。在 Aitoearn 帖子创作器中，添加您想要的主题（不带标签符号）到**主题**字段。就是这样！

![](//d33v4339jhl8k0.cloudfront.net/docs/assets/64cb6cf86d4a3e191f237e66/图片s/6881023b401e5f22412d7066/文件-lSAzM70nHy.jpg)

### 选项 2：使用文本内主题标签

您还可以通过在帖子正文中包含有效标签来添加主题。

![Aitoearn 创作器的图片，帖子中添加了内联主题 (#TennisThreads)。箭头指向帖子中的标签和帖子预览中的标签。](//d33v4339jhl8k0.cloudfront.net/docs/assets/64cb6cf86d4a3e191f237e66/图片s/688101e8c11e7d1454719293/文件-iulU0BKSxh.jpg)

**文本内标签的重要规则：**

标签必须以**标签符号（**`**#**` **）**开头。**每个帖子只允许一个主题标签**。Aitoearn 将使用找到的**第一个有效**标签。标签将在 Aitoearn 仪表板中显示时不带 `#`。**仅包含数字的标签**，如 `#1`，将被忽略（假定为数字符号）。文本内标签将在以下任何情况下自动**结束**：

- 空格或换行
- 句号 `.`
- 和号 `&`
- 在号 `@`
- 感叹号 `!`
- 问号 `?`
- 逗号 `,`
- 分号 `;`
- 冒号 `:`

**示例：**
- ✅ **您做了哪些改变来提高您的 #Tennis 发球？**（正确）
- 🚫 **我推荐的 #1 提示是...**（错误 - "#1" 不会被视为主题）

## 常见问题

### 您支持在 Aitoearn 中的 Threads 投票吗？

目前，我们不支持通过 Aitoearn 创建 Threads 投票。如果您希望我们在未来支持它们，您可以在[我们的功能请求板](https://suggestions.aitoearn.com/p/调度-polls-在-threads)上为此建议投票。

**为什么我看不到连接 Threads 的选项？**

要连接 Threads 到 Aitoearn：
- 您的账户必须在[我们的新计划](http://aitoearn.com/定价)中。
- 您的 Threads 账户必须是公开的。
- 您的 Threads 账户必须连接到一个 Instagram 账户。
