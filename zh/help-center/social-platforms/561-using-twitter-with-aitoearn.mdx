## 这些功能适用于 X/Twitter

| 发布 | 分析 | 互动 |
| --- | --- | --- |
| ✓ | ✓ | ✓ |

## X/Twitter 内容发布要求
| 内容 | 纯文字 | 图片 | 视频 |
| --- | --- | --- | --- |
| 支持情况 | ✓ | ✓ | ✓ |

| 分类                     | 视频                                                                 | 图片                  | GIF                            |
|--------------------------|----------------------------------------------------------------------|-----------------------|--------------------------------|
| **每条帖子允许数量**        | 1 个视频                                                               | 4 张图片                | 1 个 GIF                        |
| **视频时长**               | 最短 0.5 秒，最长 140 秒<br />📝 X Premium/Twitter Blue 订阅用户可上传更长视频，但仅限于在 X/Twitter 原生发布 | -                     | -                              |
| **文件大小**               | 最大 512MB                                                            | 最大 5MB               | 最大 15MB                       |
| **尺寸要求**               | -                                                                    | 最大 8192 × 8192 像素   | 最大 1280 × 1080 像素<br />总像素上限：3 亿 |
| **支持的文件格式**          | MP4, MOV, AVI, M4V<br />📝 仅音频的 MP4 文件不能上传，必须包含视频        | JPG, PNG, HEIC, WebP  | GIF                            |
| **信息流自动播放**          | 不支持                                                                | -                     | -                              |
| **自定义缩略图**            | 不支持                                                                | -                     | -                              |
| **推荐帧数**               | ≤ 60 FPS                                                             | -                     | ≤ 350 帧                        |

- 每日发帖限制	每 24 小时限发 100 条帖子。
- 字符限制	
  - X 免费用户：280 个字符
  - X Basic、Premium 和 Premium + 用户：25,000 个字符*

### GIF 额外说明
若在发布动图时出现画质下降或错误，建议遵循以下规则：
- GIF 最大分辨率：**1280 × 1080**（宽 × 高）  
- GIF 最大帧数：**350**  
- GIF 最大像素总数：**宽 × 高 × 帧数 ≤ 3 亿**
