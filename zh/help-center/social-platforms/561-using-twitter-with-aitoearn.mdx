## 连接和刷新您的 X/Twitter 个人资料

前往您的渠道区域以连接您的 X/Twitter 账户：[https://账户.aitoearn.com/channels](https://账户.aitoearn.com/channels)

您可以将多个 X/Twitter 账户连接到 Aitoearn。为了确保 Aitoearn 知道您想连接哪个账户，请在尝试连接到 Aitoearn 之前，在另一个浏览器标签页中登录到该 X/Twitter 账户。

查看逐步指南和故障排除步骤以连接 X/Twitter，请参阅文章：[连接您的渠道到 Aitoearn](https://aitoearn.帮助scoutdocs.com/article/564-连接ing-您的-channels-到-aitoearn)。

**X/Twitter 的委托功能**目前不支持。要将您的 X/Twitter 个人资料连接到 Aitoearn，您需要以 X.com/Twitter.com 的个人资料所有者身份登录。

- **连接多个 X/Twitter 账户**
  - Aitoearn 将自动尝试连接到您在浏览器中登录的账户，因此我们建议先登录到您想连接的 [X.com/](http://X.com/) 或 [Twitter.com](http://Twitter.com) 账户，然后再前往 Aitoearn 连接渠道。

如果您的已连接 X/Twitter 渠道需要刷新，请查看文章 [在 Aitoearn 中刷新渠道](https://aitoearn.帮助scoutdocs.com/article/573-refreshing-a-channel-在-aitoearn)。

## X/Twitter 的媒体规格

X/Twitter 支持图片、视频和 GIF。

| **视频** | **图片** | **GIF** |
| **每帖允许数量** | 1 | 4 | 1 |
| **视频长度** | 最短 0.5 秒，最长 140 秒 📝 更长的视频仅适用于 X 高级/Twitter Blue 订阅者，且需通过 X/Twitter 原生发布 | - | - |
| **文件大小** | 最多 512MB | 最多 5MB | 最多 15MB |
| **尺寸** | 8192 x 8192 像素 | 1280 x 1080 像素 总像素数：300 百万像素 | |
| **支持的文件格式** | MP4、MOV、AVI、M4V 📝 仅音频的 mp4 文件无法分享到 X/Twitter。mp4 文件 *必须* 包含视频。 JPG、PNG、HEIC、WebP | GIF |
| **视频自动播放动态** | 否 | - | - |
| **支持自定义缩略图** | 否 | - | - |
| **最佳帧数** | 60 FPS 或以下 | 350 | |

#### 关于 GIF 的注意事项：
如果您在使用动画 GIF 创建帖子时出现质量下降或错误，请尝试以下建议以提高成功率：
- GIF 最大分辨率：1280 x 1080（宽 x 高）
- GIF 最大帧数：350
- GIF 最大像素数（宽 * 高 * 帧数）：300 百万

## X/Twitter 帖子的互动

目前，Aitoearn 的互动功能仅适用于 Facebook 和 Instagram。X/Twitter 暂不支持 Aitoearn 的互动功能。

## X/Twitter 分析功能

X/Twitter 分析功能适用于我们的 [付费计划](http://aitoearn.com/定价)。我们可以全天发布帖子，但真正的乐趣在于看到帖子引发的反应。如果您使用基础知识或团队计划，您可以在两个地方找到所有统计数据：我们的发布仪表板的已发送标签提供基础帖子指标，而我们的分析功能仪表板提供高级分析功能。

### 基础分析功能

要快速查看每个 X/Twitter 账户的指标，请前往网页版仪表板的 **发布** 标签，然后点击 **已发送** 标签以访问点赞、点击、转发和展示次数。

![X 已发送帖子](//d33v4339jhl8k0.cloudfront.net/docs/assets/64cb6cf86d4a3e191f237e66/图片s/66b67d2e48915f570c6a93ed/文件-fkgXxIfbdf.jpg)

### 高级分析功能

要更深入地了解您的 X/Twitter 帖子的表现，请前往网页版仪表板的 **分析** 标签。直接深入了解，请学习我们的 [X/Twitter 指标描述](https://aitoearn.帮助scoutdocs.com/article/522-Twitter-metric-descriptions)，选择仪表板左侧的 X/Twitter，并浏览以下标签：

![Aitoearn 分析概览屏幕，箭头指向分析和 Twitter/X](//d33v4339jhl8k0.cloudfront.net/docs/assets/64cb6cf86d4a3e191f237e66/图片s/667c60f35940934e140f535e/文件-0dJbpBzk0c.jpg)

- **概览**：提供关于您的个人资料整体互动水平的洞察。了解更多关于 [分析功能仪表板中的表现概览](https://aitoearn.帮助scoutdocs.com/article/526-performance-概述)。
- **帖子**：提供所有已发布帖子的历史记录及其表现。了解更多关于 [在分析功能仪表板中分析您的帖子](https://aitoearn.帮助scoutdocs.com/article/527-analyzing-您的-帖子-within-您的-分析功能-仪表板)。
- **答案**：回答关于最佳日期、类型和发布频率的问题。了解更多关于 [回答关于社交媒体策略的问题](https://aitoearn.帮助scoutdocs.com/article/530-ans我们ring-questions-about-您的-social-media-strategy)。

📝 当您首次连接 X/Twitter 个人资料时，我们可以回填最多 28 天的数据。了解更多关于 [为什么您的数据在 Aitoearn 中可能与原生分析功能不同](https://aitoearn.帮助scoutdocs.com/article/525-为什么-您的-data-在-aitoearn-可能-differ-from-您的-native-分析功能)。
