## 内容发布要求

| 内容 | 纯文字 | 图片 | 视频 |
| --- | --- | --- | --- |
| 支持情况 | ✖️ | ✓ | ✓ |

| 分类 | 图文笔记 | 视频笔记 |
| --- | --- | --- |
| **支持格式** | 图片：JPG、JPEG、PNG 等常见格式 | 视频：MP4、MOV（推荐 H.264 编码），支持 WebM |
| **数量限制** | 最多 9 张图片 | 每条笔记 1 个视频 |
| **尺寸要求** | 推荐 1080×1920 像素（竖屏最佳）<br />支持宽高比 4:3 至 3:4，如 1080×810 – 1080×1440 像素 | 推荐竖屏 9:16（1080×1920 像素），支持 16:9 横屏或 1:1 方形比例 |
| **分辨率** | 建议 ≥720P，保证画质清晰 | 支持最高 1080p，超过会被压缩 |
| **时长限制** | N/A | 普通用户：≤15 分钟<br />部分创作者可上传 10 分钟以上视频 |
| **文件大小** | 单张图片建议 ≤5MB（更大可能影响加载和推荐） | 单个视频 ≤1GB；推荐 ≤300MB 以减少上传失败风险 |
| **帧率与音频** | N/A | 建议 30 FPS，音频使用 AAC 编码，码率 ≥128kbps |
| **封面图** | 首图即封面，建议竖屏 1080×1440 像素 | 支持上传自定义封面，推荐竖屏 1080×1440 像素，横屏 1440×1080 像素 |
| **文本说明** | 标题 + 笔记正文；正文建议 300–800 字，图文并茂更易获得推荐 | 视频描述建议简洁，搭配标签/话题；避免过长文本 |
