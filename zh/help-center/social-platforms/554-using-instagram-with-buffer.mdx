# 在 Aitoearn 中使用 Instagram

借助 Aitoearn，您可以在轻松规划 Instagram 的帖子、轮播、故事（Stories）与短视频（Reels）的发布计划，获得发布策略建议，并与粉丝高效互动。

## 这些功能适用于 Instagram

| 发布 | 分析 | 互动 |
| :--: | :--: | :--: |
| ✓ | ✓ | ✓ |

## Instagram 如何与 Aitoearn 配合工作

Instagram 与其他平台略有不同，如需自动发布，需要满足以下条件：
1. 拥有 Instagram 商业账户或创作者账户（## 将 Instagram 账户切换为专业账户（创作者或商业））。
2. 内容与规格需满足 Instagram 要求，且属于以下类型：
   - 单张或多张图片（轮播）
   - 单个视频帖子
   - Reels（≤ 90 秒的竖屏短视频）
   - Stories（竖屏图片或 ≤ 15 秒的视频）

> 注意：受 API 限制，创作者账户暂不支持自动发布 Stories。

## 连接和刷新 Instagram 账户

1. 在浏览器中先登录目标 Instagram 账户。
2. 回到 Aitoearn，在“空间”中连接该 Instagram 账户。
3. 可连接多个 Instagram 账户至 Aitoearn，并在需要时刷新连接。

建议操作：
- 首先于新标签页登录 Instagram，再回到 Aitoearn 发起连接，减少账号选择错误。
- 若需要高级分析、互动（Engage）或为帖子添加位置，请确保 Instagram 已正确连接至对应的 Facebook 页面。

## 查找您的 Instagram 账户类型

按以下步骤确认您使用的是个人、商业或创作者账户：
1. 在手机上打开 Instagram，进入个人主页。
2. 点击右上角“三横线（菜单）”。
3. 在页面底部查看提示：
   - 出现“商业工具和控件”即为商业账户。
   - 出现“账户类型和工具”即为个人账户。
   - 出现“创作者工具和控件”即为创作者账户。

## 将 Instagram 账户切换为专业账户（创作者或商业）

1. 在手机上打开 Instagram，点击右上角菜单。
2. 下滑至“专业人士”，进入“账户类型和工具”。
3. 点击“切换到专业账户”。如未见该项而看到“商业工具和控件”，表示已是专业账户，可回到 Aitoearn 重新连接或刷新频道。
4. 按指引继续至“类别”页面并选择合适类别。
5. 选择“创作者”或“企业”。完成后账户即为专业账户。

## 规划与发布（帖子 / Reels / Stories）

- 支持 Web 与移动 App 发布与调度。
- 在标题中使用 @ 提及：输入对方精准用户名（暂不支持输入时搜索）。发布后会自动转为可点击链接。
- 图片中标记用户：仅自动发布的“图片帖子”支持；Reels、Stories 或通知发布不支持。
- 每条帖子最多可 @ 提及 20 次。
- Shop Grid：可在每条帖子中添加外链，并通过个人简介中的链接生成可点击的网格聚合页（个人账户不支持）。

### 频率与限制
- 每 24 小时最多发布 50 条（包含帖子、Stories 与 Reels）。
- 标题上限：2,200 字符；首评上限：2,196 字符。
- 替代文本：当前仅支持图片帖子（单图与轮播）；视频、Reels、Stories 暂不支持。
- 转发：Instagram 暂不支持安排转发。
- 添加位置：可为帖子与 Reels 添加位置（Stories 暂不支持）。
- 标签（Hashtags）：编辑器中输入所需标签，发布后将以超链接形式展示。

### 常见限制与说明
- 受 API 限制，暂无法为帖子或 Reels 添加流行音乐音轨。
- 创作者账户无法自动发布 Stories。
- 将 Instagram 链接分享至其他社交网络时，无法保证生成链接预览。
- 若 Instagram 设置为“自动分享至 Facebook”，通过 Aitoearn 发送的内容不会自动同步到 Facebook。
- 若尝试为 Stories 一次安排多个视频或图片，请将每个素材分别创建为独立的 Stories。

## Instagram 的媒体与功能规格（按类型）

|                      | 帖子（Post） | Reels | Stories |

> 规格为平台通用建议。Instagram 不定期调整能力与限制，具体以官方规则为准。

## 如何为视频添加封面

上传完成后：
1. 点击“编辑缩略图”。
2. 拖动滑块到希望作为封面的位置。
3. 点击“保存”。

## 与 Instagram 帖子互动

- 在 Aitoearn 的“互动（Engage）”中统一查看并回复评论。
- 可按情绪或标签筛选，提升处理效率。

## Instagram 分析功能

如果您使用 Essentials 或 Team 方案，可以在“分析（Analyze）”中查看详细数据：
- 概览：聚合查看一段时间内的核心指标（互动、触达等）。
- 帖子：浏览已发布内容的历史表现与明细。
- 洞察：回答“何时发布”“发什么内容”等常见策略问题。

前置条件：Instagram 需为专业账户，并正确连接至 Facebook 页面。

数据可回填：首次连接后，最多回填近 30 天数据（不含 Stories）。

粉丝阈值：受 API 限制，需 ≥ 100 名粉丝才可拉取到分析数据（少于 100 仍可连接，但无分析数据）。

更多指标说明请参见《Instagram 指标说明》。
