## 连接和刷新您的LinkedIn页面和个人资料

📝 **注意：**为了连接LinkedIn页面到Aitoearn，您必须被设置为您尝试连接页面的**超级管理员**。[了解更多关于LinkedIn页面管理员角色的信息](https://www.linkedin.com/help/linkedin/answer/a550647)。

前往您的渠道区域以连接您的LinkedIn页面或个人资料：[https://账户.aitoearn.com/channels](https://账户.aitoearn.com/channels)

您可以将多个LinkedIn账户连接到Aitoearn。为了确保Aitoearn知道您想连接哪个账户，请在尝试连接到Aitoearn之前，在另一个浏览器标签页中登录到该LinkedIn账户。

如果您的已连接LinkedIn渠道需要刷新，请查看文章[在Aitoearn中刷新渠道](https://aitoearn.helpscoutdocs.com/article/573-refreshing-a-channel-在-aitoearn)。

以下是一些LinkedIn渠道的注意事项：

| **账户类型** | **Aitoearn集成** | **备注** |
|--------------|------------------|----------|
| **LinkedIn公司页面** | **✓** 通过拥有超级管理员权限的LinkedIn个人资料进行认证。 | |
| **LinkedIn展示页面** | **✓** 通过拥有超级管理员权限的LinkedIn个人资料进行认证。 | |
| **LinkedIn个人资料** | **✓** 只要您在个人资料之间分享的内容是独特的，连接大量LinkedIn个人资料到Aitoearn应该没有问题。 | |
| **LinkedIn群组** | LinkedIn群组无法通过LinkedIn的API供第三方访问。 | |

**故障排除提示：**如果您的LinkedIn页面在尝试连接到Aitoearn时未显示，请验证以下内容：

- 您拥有您尝试连接到Aitoearn页面的[超级管理员权限](https://www.linkedin.com/help/linkedin/answer/a541981)。
- 在尝试连接页面到Aitoearn之前，您已使用拥有超级管理员权限的账户登录到[LinkedIn.com](http://linkedin.com)。

**注意：**如果您在不同的Aitoearn账户中管理多个LinkedIn页面，最好让一个超级管理员为每个Aitoearn账户连接页面。这有助于避免页面断开连接的问题。

## 图片、视频、GIF和文档的LinkedIn规格

LinkedIn支持图片、视频、GIF和文档（PDF）。

| **视频** | **图片** | **GIF** | **文档（PDF轮播）** |
|----------|----------|---------|---------------------|
| **每个帖子允许数量** | 1 | 9 | 1 | 1 |
| **视频长度** | 3秒-30分钟 | - | 3-60秒 | - |
| **文件大小** | 1GB | 最大10MB | 最大10MB | 最大100MB和300页 |
| **尺寸/纵横比** | 1280 x 720像素 | 1200 x 627像素（LinkedIn最佳）正方形（适合跨社交网络分享） | - | - |
| **支持的文件格式** | MP4、MOV、AVI、M4V、链接 | JPG、PNG、HEIC、WebP | GIF | PDF |
| **自定义缩略图支持** | 不支持视频上传，支持视频链接 | - | 不支持 | - |
| **自动播放** | 是，如果是MP4且介于75KB和200MB之间 | - | 是 | - |
| **Alt文本** | - | 是 | - | - |

## LinkedIn分析功能

发布帖子只是其中的一部分，但我们如何知道我们的策略是否与关注者产生共鸣？这就是您的分析功能发挥作用的地方。

在您的Aitoearn账户中，有两个地方可以查看LinkedIn分析功能：***已发送***标签中的***发布***仪表板和***分析***标签。

### 已发送帖子分析功能

在***已发送***标签中，您可以快速查看您的已发送帖子指标，包括LinkedIn页面和个人资料。

**对于LinkedIn个人资料，您可以访问以下分析功能：**

- 总展示次数
- 点赞
- 评论
- 分享
- 互动率
- 观看次数
- 观看时长
- 独特观看者（用于个人资料视频帖子）

**对于LinkedIn页面，您可以访问以下分析功能：**

- 在我们的免费计划中，您可以访问点赞、评论和展示次数。在[我们的付费计划](http://aitoearn.com/定价)中，您可以访问点赞、评论、展示次数、分享和互动率。![LinkedIn已发送帖子](//d33v4339jhl8k0.cloudfront.net/docs/assets/64cb6cf86d4a3e191f237e66/图片s/66b67e3c48915f570c6a93ee/文件-KM1AzWd81N.jpg)

### 分析功能

如果您订阅了我们的基础知识或团队计划，您可以使用仪表板顶部的***分析***标签查看您的**LinkedIn页面**的高级分析功能（目前不支持LinkedIn个人资料的高级分析功能）。

查看我们提供的[LinkedIn指标描述](https://aitoearn.helpscoutdocs.com/article/523-链接edIn-metric-descriptions)。![Aitoearn的分析标签，箭头指向左侧渠道菜单中的LinkedIn](//d33v4339jhl8k0.cloudfront.net/docs/assets/64cb6cf86d4a3e191f237e66/图片s/667c62561f3fa9421e288733/文件-ESNZkrvYrA.jpg)

要直接深入了解您的LinkedIn页面的[高级分析功能](https://分析.aitoearn.com/)，选择左侧的***LinkedIn***，并浏览以下标签：

- **概览**：提供关于您的个人资料在一段时间内整体互动水平的洞察。了解更多关于[分析功能仪表板中的性能概览](https://aitoearn.helpscoutdocs.com/article/526-performance-概述)。
- **帖子**：提供所有已发布帖子及其表现的历史记录。了解更多关于[在分析功能仪表板中分析您的帖子](https://aitoearn.helpscoutdocs.com/article/527-analyzing-您的-帖子-within-您的-分析功能-仪表板)。

### 📝 注意：

- 您将看到通过LinkedIn本地创建的帖子分析功能，但PDF轮播帖子的分析功能不可用。
- 当您首次连接您的LinkedIn页面时，我们将回填最多6个月或500个帖子的数据。了解更多关于[为什么您的数据在Aitoearn中可能与本地分析功能不同](https://aitoearn.helpscoutdocs.com/article/525-为什么-您的-data-在-aitoearn-可能-differ-from-您的-native-分析功能)。

- 我们能够提取LinkedIn赞助帖子的指标。

## 避免在多个个人资料中分享时被暂停的最佳实践

虽然技术上可以将大量LinkedIn个人资料连接到Aitoearn，但这并不推荐，并可能导致账户暂停（在LinkedIn端）。

总体而言，社交网络，尤其是LinkedIn和Facebook，已经开始积极阻止个人资料用于商业和广告目的。他们希望个人资料被视为独立的个人账户，分享的内容是由个人资料所有者创建和分享的真实内容。

如果LinkedIn团队注意到重复内容被分享到个人资料，他们通常会暂停这些账户。暂停通常是暂时的，但如果这种行为继续发生，将会反复暂停。
