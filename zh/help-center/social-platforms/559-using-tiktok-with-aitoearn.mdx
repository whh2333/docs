## Aitoearn 中的 TikTok 工作原理

在 TikTok 上发布内容时，有两种发布方式：

***自动发布*** - 在 Aitoearn 中调度您的视频，我们会自动为您发布。

***通知发布*** - 在 Aitoearn 中调度您的视频，当需要发布您的短视频时，您会收到手机通知，您可以在 TikTok 应用中完成发布。

在编辑器中，您会看到这些选项，您可以选择 **自动发布**（如果您的帖子可以自动调度）或 **通知我**（如果您选择接收移动端通知，或者帖子包含无法自动发布的内容，例如超出时长限制的视频）。

## 连接和刷新您的 TikTok

前往您的渠道区域，连接您的 TikTok 账户：[https://账户.aitoearn.com/channels](https://账户.aitoearn.com/channels)

您可以将多个 TikTok 账户连接到 Aitoearn。为了确保 Aitoearn 知道您想连接哪个账户，请在尝试连接之前，在另一个浏览器标签页中登录到该 TikTok 账户。

## TikTok 的媒体规格

*TikTok 支持在 Aitoearn 桌面端版本中发布图片和视频。我们的移动应用目前支持 TikTok 视频发布（图片支持即将推出！）。

| **视频** | **图片*** |
| -------- | --------- |
| **每帖数量** | 1 | 最多 10 |
| **视频时长** | 3 秒至 10 分钟 | 不适用 |
| **文件大小** | 1GB 或更小，且最小高度和宽度均为 360px | 最多 20MB |
| **尺寸/宽高比** | 最小高度和宽度为 360 像素 | 最大分辨率：1080 x 1920 像素或 1920 x 1080 像素 |
| **支持的文件格式** | MOV、MP4 或 WebM | JPG 和 WebP（HEIC 和 PNG 将转换为 JPG） |
| **自定义缩略图支持** | 在 Aitoearn 编辑器中选择的帧将用作 TikTok 个人资料中的视频缩略图。 | 不适用 |
| **最佳帧数** | 23 FPS 至 60 FPS | 不适用 |
| **自动播放** | 是 | 不适用 |
| **Canva 集成** | 否 | 是 |
| **Alt 文本** | 无法通过 Aitoearn 为 TikTok 视频添加 Alt 文本。了解更多：[为图片添加 Alt 文本](https://aitoearn.帮助scoutdocs.com/article/618-添加ing-alt-text-到-您的-图片s)。由于 API 限制，不支持。 |

## 如何为您的视频添加缩略图

默认情况下，视频的第一帧（0 秒）将用作缩略图。如果您想更改缩略图，请按照以下步骤操作：
- 视频上传完成后，点击 ***编辑缩略图***。
- 拖动滑块到视频中您想用作封面/缩略图图片的位置。
- 点击 ***保存***。

## 常见问题

### 为什么我无法在通过 Aitoearn 发布的 TikTok 帖子中使用“推广”功能？

TikTok 的“推广”功能无法在通过 Aitoearn 发布的帖子中使用。要推广您的帖子，您需要通过 TikTok 原生应用发布帖子。

### TikTok 分析功能中是否包含桌面端和移动端的观看次数？

TikTok 的观看次数仅在观众登录并通过移动应用观看时计入。在桌面端或未登录状态下的观看次数不会被计入。
