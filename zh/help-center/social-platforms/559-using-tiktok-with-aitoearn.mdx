## TikTok 发布内容要求

| 内容 | 纯文字 | 图片 | 视频 |
| --- | --- | --- | --- |
| 支持情况 | x | ✓ | ✓ |

| **视频** | **图片*** |
| -------- | --------- |
| **每帖数量** | 1 | 最多 10 |
| **视频时长** | 3 秒至 10 分钟 | 不适用 |
| **文件大小** | 1GB 或更小，且最小高度和宽度均为 360px | 最多 20MB |
| **尺寸/宽高比** | 最小高度和宽度为 360 像素 | 最大分辨率：1080 x 1920 像素或 1920 x 1080 像素 |
| **支持的文件格式** | MOV、MP4 或 WebM | JPG 和 WebP（HEIC 和 PNG 将转换为 JPG） |
| **自定义缩略图支持** | 在 Aitoearn 编辑器中选择的帧将用作 TikTok 个人资料中的视频缩略图。 | 不适用 |
| **最佳帧数** | 23 FPS 至 60 FPS | 不适用 |
| **自动播放** | 是 | 不适用 |
| **Canva 集成** | 否 | 是 |
| **Alt 文本** | 无法通过 Aitoearn 为 TikTok 视频添加 Alt 文本。了解更多：[为图片添加 Alt 文本](https://aitoearn.帮助scoutdocs.com/article/618-添加ing-alt-text-到-您的-图片s)。由于 API 限制，不支持。 |

- 每 24 小时最多 25 篇内容。
- 每条内容最多 2200 个字符。
