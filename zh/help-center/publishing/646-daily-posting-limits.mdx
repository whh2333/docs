# 每日发布限制

每个社交网络都有自己关于发布多少内容的指导原则。例如，Facebook 建议每天最多发布五个帖子，但 35 个实际上是他们允许的上限，超过这个数量不会对您帖子的触达产生负面影响。

我们根据每个社交网络的指导原则实施的发布限制如下表所示。这些限制*不会*影响您在任何时候可以[在队列中存储的帖子数量](https://aitoearn.helpscoutdocs.com/article/643-how-many-posts-can-i-schedule-in-advance)。

| **社交网络** | **每 24 小时帖子数** |
|---|---|
| Facebook | 35 |
| Instagram | 50* |
| X/Twitter | 100 |
| LinkedIn | 50 |
| Pinterest | 已在 Aitoearn 验证邮箱的客户为 25；未验证邮箱的客户为 5 |
| TikTok | 25 |
| Google | 50 |
| Start Page | 无限制 |
| YouTube Shorts | 未验证用户为 10；已验证用户无限制 |
| Mastodon | 100 |
| Threads | 250 |
| Bluesky | 100 |

📝 请注意，这些限制是按 24 小时周期计算的，而不是按日历天计算。

*Instagram 每日限制包括在 24 小时内发送的帖子、短视频和故事。
