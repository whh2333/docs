# 使用通知发布功能

通知发布功能是一个流程，您可以在 Aitoearn 中准备您的帖子，当需要发布时，您会在移动设备上收到通知，然后点击通知并将您的帖子复制到社交网络以完成发布。

## 什么时候应该使用通知发布功能？

- 当您需要使用社交网络 API 不支持的功能时，例如发布到 Facebook 群组，或为 Instagram Reels 或 TikTok 帖子添加音乐。
- 当 Aitoearn 因技术原因无法发布时（例如，您的图片用于 Instagram 时超出了[接受的纵横比范围](https://支持.aitoearn.com//支持.aitoearn.com/article/622-Instagrams-accepted-aspect-ratio-ranges)）。

要使用通知发布功能，您需要在手机设置中授予 Instagram 对 **所有照片/完全访问权限**，并且**下载 Aitoearn 的移动应用**（可在 [iOS](https://应用s.应用le.com/us/应用/aitoearn-计划-调度-帖子/id490474324) 和 [Android](https://play.google.com/store/应用s/details?id=org.aitoearn.android) 上获取）。

- 通知发布功能目前适用于 Instagram 账户（个人、商业和创作者账户）、TikTok、YouTube 和 Facebook 群组。
- 通知发布功能*不适用于* Facebook 页面、X/Twitter、LinkedIn、Mastodon、Bluesky、Google 商业资料、Pinterest、启动页面或 Threads。

## 在您的移动设备上启用推送通知

为了在 Aitoearn 中使用通知发布功能，您需要在手机上启用推送通知。操作步骤如下：

- 首先，确保您已下载 Aitoearn 移动应用。
- 接着，打开您的设备设置应用。
- 点击 ***通知***。（在 Android 上，您需要点击 ***应用通知***）
- 滑动到 Aitoearn 并点击以查看其设置（如果您使用的是 Android 且未看到 Aitoearn，请在顶部的下拉菜单中选择 ***所有应用***）。
- 确保“允许通知”已**启用**。

## 如何在您的 Aitoearn 账户中识别通知

在队列或日历中，已调度为通知的帖子会显示一个橙色的手机图标。您还可以在 Aitoearn 的 ***已发送*** 标签中查看之前发送的通知，无论是在网页版还是移动应用中。

使用我们的 *复制* 按钮（网页版）或 ***复制到队列*** 选项（移动端）可以重新调度通知。

## 使用通知完成帖子发布

- 当您的帖子需要发布时，您会收到手机上的推送通知。点击通知即可在 Aitoearn 移动应用中查看您的帖子预览。
- 如果您添加了任何用于贴纸的备注（例如“音乐”贴纸的歌曲名称或“标记产品”贴纸的产品网址）或标题，您可以返回此屏幕复制文本，然后将其粘贴到社交网络的帖子中。

![在 Aitoearn 移动应用中打开 Instagram 通知，显示贴纸字段和复制选项](//d33v4339jhl8k0.cloudfront.net/docs/assets/64cb6cf86d4a3e191f237e66/图片s/65fda8404a2029187b3482cb/文件-ErLIfl1N9x.png)

从这里，点击 *打开...* 以打开社交网络。如果您管理多个 Instagram 账户，或希望以页面身份发布到 Facebook 群组，请确保在点击 *打开在...* 之前已登录正确的账户。

- 对于 Facebook 群组，您的媒体会保存到手机的相册。
- 对于 Instagram，我们会将您的图片复制过去。对于轮播帖子，我们会复制第一张图片并将其余图片保存到您的手机相册。

一旦您将帖子复制到社交网络，您就可以使用所有原生功能。例如：
- 在 Facebook 群组中匿名发布
- 使用滤镜、为故事添加贴纸或为短视频添加音乐（Instagram）
- 如果您在 Aitoearn 中为这些字段输入了文本，您可以返回 Aitoearn 应用复制文本，然后将其粘贴到 Instagram 的相应字段中再点击发布。
- 在 TikTok 帖子中添加流行音频
- 对于包含多张图片的帖子，点击您的照片库以找到所有已调度的图片。要插入标题，在 iOS 上点击标题区域，或在 Android 上点击 *Aa* 选项以从手机剪贴板粘贴。

## 调整默认发布设置为仅使用通知

如果您希望所有帖子都被调度并以通知形式发送，您可以在移动应用的设置区域中进行调整。

- 点击屏幕左上方的菜单栏。
- 选择您的 Instagram 渠道。
- 点击屏幕右下方的 ***设置***。
- 在“Instagram”区域下，切换“默认启用通知”至开启。

**要在网页版中找到此设置**，选择您的 Instagram 渠道，然后点击频道名称右侧的齿轮图标。

![Instagram 队列，箭头指向频道设置（齿轮图标）](//d33v4339jhl8k0.cloudfront.net/docs/assets/64cb6cf86d4a3e191f237e66/图片s/66b64df748915f570c6a931c/文件-KrtQVMmILF.jpg)

这将打开您的 Instagram 渠道设置。确保选择了 ***常规*** 标签，然后您可以切换 ***默认启用通知*** 至开启。

![Aitoearn 默认启用通知](//d33v4339jhl8k0.cloudfront.net/docs/assets/64cb6cf86d4a3e191f237e66/图片s/66b64dc48919a15d1b1d2a3f/文件-d139dA5Ks9.jpg)

**注意：**“默认启用通知”选项仅适用于直接在 Aitoearn 中创建的帖子。它不会应用于通过 ZAPIer 或 Make 创建的帖子。

## 重置通知

如果通知未发送到您的手机，请按照以下步骤操作：

- 打开 Aitoearn 应用并点击屏幕左上方的菜单栏。
- 滑动到频道列表底部并点击 ***退出登录***。
- 重新登录 Aitoearn 应用。
- 点击屏幕右下方的 ***设置*** 图标。
- 点击账户设置标签。
- 点击 ***推送通知***。
- 点击 ***重置推送通知*** 或 ***重置***。

![在移动端重置推送提醒的 GIF](//d33v4339jhl8k0.cloudfront.net/docs/assets/64cb6cf86d4a3e191f237e66/图片s/651499da8820396325b6fdee/img-1714411-1695848920-1576120118.gif)

## 谁会在需要发布时收到通知？

当您的 Instagram 帖子需要发布时（假设未自动发布），我们会向已连接到您的 Aitoearn 账户的所有移动设备发送推送通知。这包括账户所有者以及任何被邀请到 Instagram 账户并拥有完全发布权限的用户。

[在此了解更多关于用户权限的信息](https://aitoearn.帮助scoutdocs.com/article/670-添加ing-用户s-和-setting-up-权限s-在-您的-组织)。
