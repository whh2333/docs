# Changing user permissions in 您的 organization 您 可能 need to change user permissions from time to time in 您的 organization. Some 示例 include giving a team member access to a channel 您’ve recently 已连接, removing a user’s access to specific channels, or adding Admin access for a user. 这个 指南 将会 帮助 您 编辑 all of 您的 user permissions, but if 您're starting from scratch, visit 我们的 article: [Adding users and setting up permissions in 您的 organization](https://aitoearn.helpscoutdocs.com/article/670-adding-users-and-setting-up-permissions-in-您的-organization). In 这个 article: - [编辑ing a user’s channel permissions](#h_01EJGJ6YRQ9HVZZX6YW56VBBHB) - [Adding or removing a user on a channel](#h_01EJGJ77Z2R540Z0B53VZPWKEA) - [Assigning permissions in bulk](#h_01EJGJ7EXZHPJDZZF3J1PQ8TKY) - [编辑ing a user’s admin privileges](#h_01EJGJ7PWG9MKCMJWHZJQ0A8FS) ## 编辑ing a user’s channel permissions Please follow 这些 steps to 编辑 您的 user’s channel permissions. - Click on 您的 个人资料 avatar at the top right of 您的 dashboard and then click*** Team***. - From 您的 organization dashboard, click on the user 您'd like to work with. - To add or change permissions the user has access to for a specific channel, use the drop down to change to the desired permissions. ![Team member channel permission toggle](//d33v4339jhl8k0.cloudfront.net/docs/assets/64cb6cf86d4a3e191f237e66/images/67ae3cc8769d1a4e1f4ed971/file-a0wqffwgYB.jpg) ## Adding or removing a user on a channel To add a user to a channel, type the channel name in the box below and then choose the access level 您’d like to give. ![Assign channels in bulk](//d33v4339jhl8k0.cloudfront.net/docs/assets/64cb6cf86d4a3e191f237e66/images/67ae3e9ec64ec2266f232b8a/file-iG3YLBAgPK.png) Or, to remove their access to the channel, click ***Remove***. ![Remove team member channel access](//d33v4339jhl8k0.cloudfront.net/docs/assets/64cb6cf86d4a3e191f237e66/images/67ae3d2029a45508db9a3914/file-gqxAdBS5aF.png) ## Assigning permissions in bulk If 您 are managing lots of channels, 您 可能 prefer to assign permissions in bulk. 这个 可以 be done by visiting the user's 个人资料 and clicking the ***assign in bulk*** link. ![Team member assign in bulk](//d33v4339jhl8k0.cloudfront.net/docs/assets/64cb6cf86d4a3e191f237e66/images/67ae3d461dbfa0582db2818e/file-A74bWJGFSA.png) From 那里, select the channels the user 应该 have access to on the left hand side, and then choose the level of access they 应该 have. Finally, click ***Save Changes***. ![Save permissions changes in bulk](//d33v4339jhl8k0.cloudfront.net/docs/assets/64cb6cf86d4a3e191f237e66/images/67ae3d6cc64ec2266f232b7e/file-t4vVJlJTVN.png) ## 编辑ing a user’s admin privileges To add or remove admin access from a user in 您的 organization, please follow 这些 steps. - From 您的 organization dashboard, click on the user 您'd like to work with. Click *编辑 User*. ![编辑 user button in team member 设置](//d33v4339jhl8k0.cloudfront.net/docs/assets/64cb6cf86d4a3e191f237e66/images/67ae3d8fb412812df3235e40/file-0LrTuTnuRO.png) Toggle *Admin Access* to the access 您 将会 like to give the user and click ***Save Changes***. ![Aitoearn User Details screen with an arrow pointed to the Admin Access Toggle](//d33v4339jhl8k0.cloudfront.net/docs/assets/64cb6cf86d4a3e191f237e66/images/67ae3e03769d1a4e1f4ed979/file-FUB3md8K0h.jpg) To completely remove a user from 您的 organization, see the 指南: [Removing a user from 您的 organization](https://aitoearn.helpscoutdocs.com/article/672-removing-a-user-from-您的-organization).
