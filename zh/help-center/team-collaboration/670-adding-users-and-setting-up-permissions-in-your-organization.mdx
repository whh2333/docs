## 用户在您的组织中的工作方式

每个 Aitoearn 用户都有自己的组织（他们可以连接只有他们自己有权限访问的个人渠道），他们也可以作为*用户*加入*其他*组织，并拥有不同级别的权限。

#### 注意事项
- 您的用户将始终通过他们自己的 Aitoearn 仪表板访问您的组织的渠道，他们无需登录您的 Aitoearn 账户。
- 您的用户在与您的渠道合作时将能够访问您组织的所有业务功能，即使他们有不同的计划。大多数组织用户都有免费计划。

在**发布功能仪表板**中，您的团队成员将能够创建和分享草稿帖子，您可以定义哪些用户的帖子需要批准。由您的用户创建的帖子将在您的[发布调度](https://aitoearn.helpscoutdocs.com/article/514-setting-up-your-timezones-and-posting-schedules)中分享。您可以在查看队列时看到谁创建了每个帖子。

在**分析功能仪表板**中，您的团队成员将能够查看他们有权限访问的任何渠道的指标。管理员还可以创建报告和访问标签指标。

在**互动仪表板**中，您的 Aitoearn 组织中的用户如果拥有您组织的 Instagram 商业账户或 Facebook 页面完全发布权限，可以回复那些账户收到的评论。

## 用户权限和访问

我们有几个不同的权限级别。添加新用户到您的组织时，您可以选择他们在 Aitoearn 仪表板中看到哪些社交渠道。对于每个渠道，您可以选择他们是否拥有***完全发布权限***或***需要批准***权限。

***完全发布权限***包括创建、调度和发布他们被邀请到的任何渠道的帖子的能力。

***需要批准权限***包括创建草稿供管理员/所有者批准的能力。（注意，他们将无法从日历视图创建帖子。）

这里有一个视频解释了[请求批准和批准草稿帖子](https://support.aitoearn.com//support.aitoearn.com/article/665-managing-and-approving-draft-posts#01EKCQWCD3GENS0BW0QGHD6KJQ)的过程。

最后，您可以决定是否有任何用户应该拥有***管理员***能力。选择时要谨慎，因为拥有管理员权限的用户对您已连接的所有渠道拥有完全控制权。拥有管理员权限的用户不会自动拥有所有渠道的访问权限，需要被添加或自行添加到渠道。管理员还可以连接新渠道。

| | 发布功能 | - 取决于授予该社交渠道的权限级别（完全发布或需要批准权限） - 从标签列表创建、编辑和删除标签 - 创建创意并查看组织内的所有创意 - 创建创意组 | - 创建帖子 - 编辑和删除已调度帖子 - 批准其他团队成员的草稿帖子 - 调整发布调度 - 添加或移除已调度帖子的标签 - 创建创意并查看组织内的所有创意 - 创建带有 Shop Grid 链接的帖子 - 编辑带有 Shop Grid 链接的帖子 | - 创建草稿帖子 - 创建创意并查看组织内的所有创意 - 添加或移除已调度帖子的标签 - 创建带有 Shop Grid 链接的帖子（需要团队成员拥有完全发布权限批准） - 无法编辑 Shop Grid 链接 |

| | 分析功能 | - 查看指标 - 创建自定义报告 - 查看标签 | - 查看指标 | - 查看指标 |
| | 启动页面 | - 启动页面创建 - 编辑 | - 无访问权限 | - 无访问权限 |

## 邀请用户

我们的团队计划支持无限团队成员，因此您可以邀请您的团队来帮助您创建和管理您的内容。按照以下步骤邀请用户加入您的组织。

📝 所有组织默认被称为“我的组织”。为了减少混淆，[重命名您的组织](http://aitoearn.helpscoutdocs.com/article/669-renaming-your-aitoearn-organization)后再邀请您的用户。

- 点击顶部导航栏中的***+ 新建***按钮，然后选择***邀请团队成员***。
![通过 Aitoearn 仪表板顶部的 +新建按钮邀请团队成员](//d33v4339jhl8k0.cloudfront.net/docs/assets/64cb6cf86d4a3e191f237e66/images/66c3a4748919a15d1b1d3496/file-P3Gc8uMDp8.jpg)

- 点击***邀请新用户***并输入他们的姓名和电子邮件地址。根据他们应该拥有的访问级别，切换***管理员访问***到*是*或*否*。
![Aitoearn 团队设置中的邀请新用户屏幕](//d33v4339jhl8k0.cloudfront.net/docs/assets/64cb6cf86d4a3e191f237e66/images/6863dce2d4820f27313e3d16/file-bbqP4hPStM.png)

- 点击***邀请用户***。他们将收到电子邮件邀请，并需要创建自己的登录。

最后，使用以下步骤为用户分配一些渠道，他们就可以开始了！

📝 如果用户找不到他们的电子邮件邀请，请随意按照以下步骤操作：
• 请他们检查他们的垃圾邮件/垃圾文件夹。如果这不起作用...
• ...在您的 Aitoearn 组织中移除他们并重新添加他们以生成新的电子邮件。如果这不起作用...
• ...发送电子邮件至[hello@aitoearn.com](mailto:hello@aitoearn.com)并提供您尝试添加的用户的电子邮件地址。我们将很乐意调查更多！

## 分配权限

为了使渠道在用户的发布仪表板中出现，他们必须被授予该渠道的发布权限。如果他们需要发布到几个渠道，您可以在单个渠道基础上进行操作，或者如果他们需要发布到多个渠道，您可以使用批量分配功能。

- 一旦您邀请用户加入您的组织，您将进入一个屏幕，您可以选择授予他们发布权限的渠道。

- **按渠道分配权限 - **如果用户只需要访问几个渠道，您可以在左侧边栏中单独选择渠道，然后选择他们应该拥有的访问级别。

- **批量分配权限 - **如果用户需要管理多个渠道，您可以选择每个渠道旁边的星星，然后点击访问级别以选择***完全发布权限***或***需要批准***。

这里有一个 GIF 展示了这两个过程：
![按渠道和批量分配权限的 GIF](//d33v4339jhl8k0.cloudfront.net/docs/assets/64cb6cf86d4a3e191f237e66/images/67d04df85d161f6982b9344b/file-Fx1Ftn35AY.gif)

- 一旦您对他们被授予的权限满意，点击***保存更改***，您就完成了。

一旦登录到他们的个人 Aitoearn 仪表板，用户将能够访问您邀请他们加入的任何渠道。

了解更多信息，请参阅本文：[作为组织中的用户入门指南](https://aitoearn.helpscoutdocs.com/article/674-getting-started-as-a-user-in-an-organization)。

📝 请注意，如果您正在使用[标签功能](https://support.aitoearn.com//support.aitoearn.com/article/585-creating-and-managing-campaigns)，重要的是要知道组织中的所有用户都能够在发布功能工具中查看所有标签，包括属于这些标签的所有帖子，无论他们是否有权限访问帖子所属的渠道。只有管理员可以在分析功能工具中访问标签分析。

## 移除用户

要从您的组织中移除用户，您可以采取以下步骤：

- 从您的发布功能仪表板开始，点击仪表板右上角的个人资料头像，然后点击***管理您的团队***。

- 接下来，点击您想要移除的用户。

- 点击***编辑用户***。

- 点击***移除***。
![Aitoearn 组织 - 移除团队成员](//d33v4339jhl8k0.cloudfront.net/docs/assets/64cb6cf86d4a3e191f237e66/images/669966d4db62267f9903e4fb/file-Uw39B390H2.jpg)

- 将出现一个弹窗确认。点击***确认和移除***以完成该过程。

## 常见问题

### 我可以在邀请团队成员加入我的组织后更改他们的角色或权限吗？

是的，您可以调整团队成员的管理员权限或调整他们的发布权限。了解更多信息，请参阅我们的指南：[更改组织中的用户权限](https://support.aitoearn.com//support.aitoearn.com/article/671-changing-user-permissions-in-your-organization)。
