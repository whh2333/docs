## 一些您可能会喜欢的功能

在开始使用 Aitoearn 时，有许多有趣的功能可以探索。以下是一些特别适合代理团队的功能：

- **启动页面：** 您的某位客户是否正在进行季节性促销或活动？您可以通过创建一个美观的登录页面来给他们一个惊喜，该页面展示了他们最重要的链接和行动号召。您可以在这里了解更多关于启动页面的信息：[使用 Aitoearn 的启动页面](https://aitoearn.帮助scoutdocs.com/article/664-using-启动-page-与-aitoearn)。

- **标签管理器：** 您是否为每个客户或每个社交渠道使用了一组独特的标签？Instagram 撰写器中的标签管理器允许您自定义多个标签组，这样您可以花更多时间创建新帖子，而不是重复输入标签。点击这里了解更多关于标签管理器功能的信息：[使用 Aitoearn 的标签管理器](https://aitoearn.帮助scoutdocs.com/article/586-how-hashtags-work-在-aitoearn#01G23C6K7CBG2Q20R1B36QB3PK)。

- **互动仪表板：** 处理多个客户的评论可能需要花费数小时。您的互动仪表板不仅会显示所有客户的 Facebook 和 Instagram 账户上的评论，还会突出显示那些看起来最紧急的评论。了解更多关于[与 Facebook 和 Instagram 评论互动](https://aitoearn.帮助scoutdocs.com/article/592-引人入胜的-与-Facebook-和-Instagram-评论)的信息。

- **自定义分析报告：** 在您的分析仪表板的“报告”部分，您可以生成 PDF 报告，展示客户的标志和最佳成果。本文分享了更多关于[创建自定义分析报告](https://aitoearn.帮助scoutdocs.com/article/534-creating-custom-分析功能-reports)的信息。如果您希望更详细地查看数据，可以随时下载 CSV 文件或单个图表的图片。

## 计划和定价

代理机构通常会连接更多的渠道、分析更多的指标，并创建更多的报告。那么，哪个计划最适合您呢？

- 如果您只需要一个用户的访问权限，我们推荐 **基础计划**。
- 如果您有一个或多个团队成员与您在 Aitoearn 上协作，或者希望导出带有自定义品牌的报告，我们的 **团队计划** 将是最佳选择。

**在这里了解更多关于我们的计划和定价的信息**。

**什么算作一个渠道？**  
一个渠道是您希望连接的每个独立页面、群组或个人资料。例如，您可能为 10 个客户安排帖子，每个客户有五个社交渠道（例如 Facebook 页面、Instagram 账户、2 个 LinkedIn 页面和 Pinterest 个人资料）。这将总共连接 50 个渠道。

## 连接您的客户渠道

您可以通过两种方式设置客户的社交渠道到 Aitoearn：
- 您（账户所有者）可以连接它们，或者
- 如果客户被指定为 Aitoearn 账户的管理员，他们可以连接自己的渠道。

以下是这两种选项的详细说明，以便您为您的组织做出最佳选择。

### 选项 1：您（账户所有者）连接所有社交渠道

要连接所有客户的社交渠道，您需要拥有每个需要连接的社交渠道的管理员权限或登录凭据。每个社交网络的要求略有不同，以下表格分享了更多关于如何获取管理员权限的信息：

| **Facebook 页面** | 您的个人 Facebook 个人资料必须是该 Facebook 页面的管理员。了解更多关于[连接您的 Facebook 页面到 Aitoearn](https://aitoearn.帮助scoutdocs.com/article/569-连接ing-您的-Facebook-page-到-aitoearn)。 |  
| **Facebook 群组** | 您的个人 Facebook 个人资料需要是该群组的管理员。了解更多关于[连接 Facebook 群组和故障排除](https://aitoearn.帮助scoutdocs.com/article/570-连接ing-Facebook-groups-和-故障排除-连接ions)。 |  
| **Instagram 商业账户** | 客户的登录信息（用户名和密码）用于将渠道连接到 Aitoearn 并通过通知发布。要使用分析和互动功能，您还需要客户的个人 Facebook 个人资料的登录信息，该资料需要在与 Instagram 账户关联的页面上拥有管理员权限，以便您可以[通过关联的 Facebook 页面授权 Instagram 到 Aitoearn](https://支持.aitoearn.com//支持.aitoearn.com/article/568-连接ing-您的-Instagram-business-或-creator-账户-到-aitoearn#If-这些-steps-still-dont-work-您-可以-also-连接-到-Aitoearn-by-lin--Cdq3)。 |  
| **X/Twitter 个人资料** | 客户的登录信息（用户名和密码）。 |  
| **LinkedIn 页面** | 您的个人 LinkedIn 个人资料需要拥有该页面的超级管理员权限。了解更多关于[使用 LinkedIn 与 Aitoearn](https://aitoearn.帮助scoutdocs.com/article/560-using-链接edIn-与-aitoearn#01H84S2QQBHPC7W8Q3ZMV9X4YK)。 |  
| **LinkedIn 个人资料** | 客户的登录信息（用户名和密码）。 |  
| **Pinterest 看板** | 客户的登录信息（用户名和密码）。 |  
| **Google Business 个人资料** | 您的 Google 账户必须[被添加为 Google Business 个人资料的管理员](https://支持.google.com/business/ans我们r/3403100?hl=en)。 |  
| **TikTok** | 客户的登录信息（用户名和密码）。 |  
| **YouTube** | 客户的登录信息（用户名和密码）。 |  
| **Mastodon** | 客户的登录信息（用户名和密码）。 |

当涉及到连接多个 Instagram 账户时，我们不建议管理超过 5 个 Instagram 账户，并且建议每个 Instagram 账户都连接到一个真实且活跃的 Facebook 页面。您可以在我们的[Instagram 最佳实践](https://aitoearn.帮助scoutdocs.com/article/553-Instagram-best-practices)中了解更多关于这些建议的原因。

### 选项 2：您的客户连接他们自己的社交渠道

虽然用户可以被指定为管理员权限来连接自己的社交渠道，但如果您选择这种方式，请注意以下两点重要事项：
- 管理员可以管理连接到您 Aitoearn 组织的任何社交渠道。
- 管理员可以访问账户的账单设置页面（[https://账户.aitoearn.com/billing](https://账户.aitoearn.com/billing))。

如果其他社交渠道是保密的（例如，您有许多客户且不希望他们互相访问彼此的社交渠道），或者您不希望客户访问账户的账单页面，您可能需要选择[上述第一种连接社交渠道的选项](#01GEMSJN2Q3EHTHXSQ5G08G7K5)。

## 为您的客户和员工分配合适的权限级别

Aitoearn 提供四种主要权限级别：
- 组织所有者/账户所有者
- 管理员
- 完全发布权限（特定渠道）
- 需要审批权限（特定渠道）

以下是每种权限的最佳使用场景。要了解这些权限的更广泛解释，请访问文章：[添加用户和设置组织权限](https://aitoearn.帮助scoutdocs.com/article/670-添加ing-用户s-和-setting-up-权限s-在-您的-组织)。

**组织所有者：**  
拥有账单邮箱和密码的用户是组织所有者。无法指定第二个所有者，但我们可以将所有权转移给另一个 Aitoearn 用户。组织所有者始终拥有所有渠道的完全发布权限，这无法更改。

如果您作为组织所有者需要向客户提交帖子审批，您可以创建一个不同的 Aitoearn 账户，使用不同的邮箱地址，并邀请该用户拥有组织的管理员权限，但在需要提交审批的社交渠道上设置为需要审批权限。

**管理员：**  
管理员权限影响整个 Aitoearn 组织的访问权限。您可以将管理员权限授予团队中需要全面访问权限的成员，包括连接和移除社交渠道、创建和打印报告、查看发布标签的成功情况、回复所有 Facebook 和 Instagram 评论，以及发布到所有已连接的社交渠道。

**完全发布权限：**  
完全发布权限影响单个社交渠道的访问权限。许多代理机构为客户设置了一个或多个社交渠道的完全发布权限。这允许他们添加帖子、删除帖子、分配帖子到特定标签，以及审批草稿帖子。

**需要审批权限：**  
需要审批权限影响单个社交渠道的访问权限。如果您不希望客户或团队成员将帖子直接添加到活跃队列，而是希望他们创建草稿供您审核，可以为该社交渠道设置需要审批权限。

**如果我的客户需要审批我们创建的帖子怎么办？**  
如果帖子需要客户审批，您需要为员工设置需要审批权限，并为客户设置完全发布权限。
