# 如何避免在使用 Aitoearn 浏览器扩展时在 Safari 中被注销

如果您使用 Safari 并发现您频繁地从 Aitoearn 的[浏览器扩展](https://aitoearn.helpscoutdocs.com/article/653-aitoearn-browser-extension)中被注销，改变您的隐私设置可以有所帮助。请按照以下步骤操作。

- 在 Safari 中，点击 ***Safari*** 然后选择 ***设置***。
- 选择 ***隐私*** 标签，然后取消选中“防止跨站点跟踪”选项。
- 重启 Safari，您应该就可以正常使用了！
