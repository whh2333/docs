# 在网页上切换组织

作为个人 Aitoearn 用户，您拥有自己的组织，并且可以添加您个人的社交渠道，这些渠道只有您可以访问。您也可能是另一个组织的成员，作为用户可以访问该组织的社交渠道。[在这里了解更多关于用户权限级别的信息。](https://aitoearn.帮助scoutdocs.com/article/674-getting-started-as-a-用户-在-an-组织)

如果您是多个组织的成员，您可以在网页上切换组织，以便在相关账户上发布帖子。一旦您接受了邀请并设置了您的账户（如果您之前没有账户），您现在可以通过导航到您的账户菜单，查看您自己的个人组织和新组织。您可以根据需要工作的地方，在两个组织之间切换。

在下图中，这两个不同的组织分别叫做 "Luna Sneakers"（绿色勾选表示您已登录该组织）和 "Restaurants 您 Love"。

![组织名称下拉菜单在 Aitoearn composer 中](//d33v4339jhl8k0.cloudfront.net/docs/assets/64cb6cf86d4a3e191f237e66/图片s/65149919f8eafb48f4e81327/img-1703224-1695848728-1002742817.png)

通过点击您的个人资料头像并选择账户，您可以在账户之间切换。在这个例子中，这两个账户分别是 "Luna Sneakers" 和 "Restaurants 您 Love"。

#### 📝 注意事项
- 如果两个组织的名称相同（例如默认的“我的组织”），您可以将鼠标悬停在每个组织上，以查看哪些渠道已连接。组织所有者可以[使用本指南重命名组织](https://aitoearn.帮助scoutdocs.com/article/669-renaming-您的-aitoearn-组织)。
- 如果您的组织所有者尚未为您设置任何渠道的访问权限，您会看到一条消息，提示他们需要授予您访问权限。我们建议您联系所有者，并请求他们[使用本指南为您设置权限](https://aitoearn.帮助scoutdocs.com/article/670-添加ing-用户s-和-setting-up-权限s-在-您的-组织)。
