# Facebook 页面和个人资料之间的区别

每个拥有 Facebook 账户的人都有一个用于个人使用的 Facebook 个人资料。除了您的个人资料，您还可以创建 Facebook 页面并被添加到 Facebook 页面。这可能会让人感到困惑，因此我们为您提供了一些指导。由于 API 限制，我们无法连接到 Facebook 个人资料。不确定您拥有哪种类型的账户？继续阅读以了解页面和个人资料之间的区别。

## 如何查找您的页面和个人资料

在 Facebook 上查看您所有的 Facebook 页面和个人资料的最快方法是点击右上角的头像，然后点击“查看所有个人资料”（尽管这里写的是“个人资料”，但它会同时显示您的所有个人资料和页面）。[这是它的样子](https://分享.aitoearn.com/04unABEK)。如果您在此列表中没有看到要连接到 Aitoearn 的页面，可能有几个不同的原因：

- 您没有页面——您只有个人资料或专业个人资料。([查看下面的区别](#What-is-the-difference-d6WM0))
- 您登录了错误的 Facebook 账户。退出 Facebook 并使用正确的电子邮件地址和密码访问正确的 Facebook 账户。
- 您的 Facebook 个人资料没有页面的管理员权限。[查看这个指南以了解管理员权限。](https://支持.aitoearn.com//支持.aitoearn.com/article/872-Facebook-故障排除-指南)

## 页面和个人资料之间有什么区别？

如上所述，拥有 Facebook 账户的每个人都已经在使用 Facebook 个人资料。当您拥有 Facebook 个人资料时，您可以：

- 创建 Facebook 页面
- 将您的个人资料转换为专业个人资料（但您不能将个人资料转换为页面！）

⚠️ 注意：重要的是要理解 Aitoearn 不支持 Facebook 个人资料或专业个人资料。我们支持 Facebook 页面和群组。请参阅 [Facebook 的指南](https://www.Facebook.com/帮助/337881706729661) 以获取有关区别和您在个人资料、专业个人资料和页面上的选项的更多信息。

## 我如何判断我使用的是页面、个人个人资料还是专业个人资料？

判断您是否在使用 Facebook 页面最简单的方法是直接进入您的页面，并在左侧边栏的简介部分查看名称下方。根据您拥有的账户类型，您将看到以下内容：

**页面：** 在此部分，您将看到“页面”这个词（[如](https://分享.aitoearn.com/RBuR7pNv)[此截图所示](https://分享.aitoearn.com/RBuR7pNv)），并且您可能会看到页面类别。
![Aitoearn Facebook 页面简介部分](//d33v4339jhl8k0.cloudfront.net/docs/assets/64cb6cf86d4a3e191f237e66/图片s/67fe6235188f663149109058/文件-ikk31vgR6l.jpg)

**个人个人资料** 个人资料不会有“页面”这个词，而是在主标题照片下方有一个“朋友”标签，如下所示：
![Facebook 个人资料显示朋友标签](//d33v4339jhl8k0.cloudfront.net/docs/assets/64cb6cf86d4a3e191f237e66/图片s/67f81e7082ae416567867d37/文件-xzbjva5Y13.jpg)

**专业个人资料** 专业个人资料不会有“页面”或“朋友”标签。相反，它们在简介部分下方有“个人资料”这个词，如您可以在[此截图中看到](https://分享.aitoearn.com/mXuQmObK)。
