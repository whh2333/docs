## 分析功能概述

<a id="analytics-overview"></a>

使用 Aitoearn 分析功能，您可以选择一目了然地查看所有社交媒体渠道的帖子表现，或专注于特定的社交媒体渠道。您可以通过["分析"标签](https://analyze.aitoearn.com/)找到高级分析功能。

在分析仪表板中，您的团队成员将能够查看他们有权访问的任何渠道的指标。管理员还有创建报告和访问活动指标的额外能力。

![分析功能图片](//d33v4339jhl8k0.cloudfront.net/docs/assets/64cb6cf86d4a3e191f237e66/images/661ff33ee3488c2f1a5b0893/file-0IsraWyshv.jpg)

📝 分析仪表板支持 **Facebook 页面、Instagram 商业账户、LinkedIn 公司页面和展示页面，以及 X/Twitter 档案**的分析。分析仪表板不支持 Facebook 个人档案、Facebook 群组、Instagram 个人档案、LinkedIn 个人档案、Pinterest 账户、TikTok 账户、YouTube Shorts、Google 商业档案、Mastodon、Threads 或 Bluesky 的集成。

### 所有社交媒体渠道概览

我们的"主页"将所有渠道整合到一个屏幕中，让您能够获得帖子表现的高层次概览。当您登录 Aitoearn 并首次点击"分析"标签时，"主页"就会显示。

### 单个渠道指标

要查看特定社交媒体渠道的指标：

- 在 Aitoearn 左侧选择一个社交网络
- 然后，从屏幕顶部的下拉菜单中选择一个社交媒体渠道

如果您的任何社交网络只有一个渠道，它将自动被选中。但是，如果您有多个渠道（例如多个 Instagram 账户），您将看到它们都列在下拉菜单中，您可以选择一个来查看其分析数据。

![Aitoearn 分析选择渠道步骤](//d33v4339jhl8k0.cloudfront.net/docs/assets/64cb6cf86d4a3e191f237e66/images/661ff616734e0c0c779a5cf9/file-LMpW4ePB3Z.jpg)

## 创建和导出自定义报告

<a id="custom-reports"></a>

浏览分析数据后，您可以选择要添加到自定义报告中的图表和表格。要将图表或表格添加到报告中，请点击报告上方的加号（+）按钮。

点击加号后，输入您想要为新报告使用的名称，然后点击"创建报告"。或者，通过点击报告名称将表格或图表添加到现有报告中。
