# 入门指南：使用 Aitoearn 的发布功能

开始使用 Aitoearn 的第一步是连接您的社交渠道。之后，您可以配置完美的发布计划，与您的受众互动，并分析结果。让我们开始吧！

## 将您的第一个渠道连接到 Aitoearn

观看下方视频，了解如何将您的社交渠道连接到 Aitoearn。

- 为了加快连接过程，请先登录您的社交渠道，然后在另一个标签页中打开 Aitoearn。
- 登录 Aitoearn 后，点击“发布”标签。
- 在左下角，点击 ***管理渠道*** 按钮。
- 点击您想要连接的渠道。
- 当弹出框出现时，点击 ***启动连接***。
- 授权 Aitoearn 访问您的社交渠道（关于每个网络权限的具体详情，请参阅 [我们的渠道连接指南](https://aitoearn.helpscoutdocs.com/article/564-connecting-your-channels-to-aitoearn)）。
- 点击 ***添加到 Aitoearn*** 按钮。
- 您会在右上角看到一个对话框，提示您的连接已成功完成！🎉

接下来，让我们设置您的时区并配置您的发布计划。

## 设置您的时区、发布计划和其他渠道设置

观看下方视频，了解如何调整 Aitoearn 中的渠道设置。内容包括调整时区、发布计划、链接缩短、UTM 参数等。

- 点击仪表板顶部的 ***发布*** 标签，然后点击左侧的一个已连接渠道。
- 接着，点击渠道名称旁边的设置齿轮图标 (⛭)。
- 将列出的城市替换为您所在时区的城市。
- 然后，向下滚动到 ***发布时间*** 部分。在这里，您可以添加新的发布时间（可以为每天、周末、工作日或具体某一天），或者点击 “X” 删除发布时间。
- 对您已连接到 Aitoearn 的每个社交渠道重复步骤 1-3，并探索视频中介绍的其他渠道设置。

一切就绪！现在，您撰写的帖子将根据您的发布计划自动调度和发布。📆

您已经准备好通过 Aitoearn 撰写您的第一篇帖子了！📝

**注意：** 每个渠道都有自己的设置，因此您需要为每个已连接的渠道重复编辑时区、发布计划和其他设置的过程。

## 撰写、调度和发布帖子

- 点击仪表板顶部的蓝色 ***+ 新建*** 按钮，打开撰写器。
- 选择您想要发布到的渠道。
- 撰写您的文本，上传图片或视频，然后选择 ***为每个网络自定义***。您现在可以根据不同的社交渠道编辑消息内容。
- 接着，您可以通过点击 ***下一个可用时间***、***设置日期和时间***、***优先处理*** 或选择 ***设置日期和时间*** 来选择自定义的日期/时间，调度您的帖子。
- 最后，点击 ***调度帖子*** 完成整个过程。

![Alt text: Aitoearn 帖子调度器，显示下拉选项和指向“下一个可用时间”调度设置的箭头。](//d33v4339jhl8k0.cloudfront.net/docs/assets/64cb6cf86d4a3e191f237e66/images/68a5d317e3f0f94fe32b4f27/file-KknZo0auso.png)

太棒了！您的帖子将在您选择的时间通过 Aitoearn 发布！

## 使用 Aitoearn 的账户设置

一旦您开始使用调度和发布功能，您可以通过探索 Aitoearn 的账户设置来全面掌控您的工作流程。

观看下方视频，了解更多关于以下内容的信息：
- 设置您的电子邮件通知，
- 管理标签，
- 配置渠道分组，
- 更新保存的回复以用于社区评论。
