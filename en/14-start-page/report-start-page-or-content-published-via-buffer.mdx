<h2 id="h_01HC03JPHXH68HJQWZ18ANSJ6Y">How to report Start Page or other content published via Buffer</h2>
<p>If you've come across a Start Page or content published via Buffer that you perceive to violate our <a href="https://buffer.com/terms" target="_blank" rel="noopener">Terms of Use</a>, we'd be grateful if you would report the content to us via <a href="https://buffersurvey.typeform.com/to/r4HrjZ0t" target="_blank" rel="noopener">this content report form</a>.</p>
<p>You can also access this form directly from the Start Page by clicking the menu icon (three dots) next to "Powered by Buffer" and then clicking&nbsp;<em><strong>Report this Start Page.</strong></em></p>
<p><img alt="Report a Start Page" src="//d33v4339jhl8k0.cloudfront.net/docs/assets/64cb6cf86d4a3e191f237e66/images/651498e2556a3e3cf18cbfcd/img-1705646-1695848673-1892281819.png" style="width: 427.986px; max-width: 100%; "></p><br>
<p>Examples of content that might violate our Terms:</p>
<ul>
 <li>intellectual property infringement;</li>
 <li>falsely Impersonating some other person, organization or entity;</li>
 <li>false or misleading information presented as truth;</li>
 <li>hate speech, abusive or harmful content;</li>
 <li>astroturfing, phishing, and spam;</li>
 <li>etc</li>
</ul>
