        <h1 class="title">Changing user permissions in your organization</h1>
    
        <a href="javascript:window.print()" class="printArticle" title="Print this article" aria-label="Print this Article"><i class="icon-print"></i></a>
        <p>You may need to change user permissions from time to time in your organization. Some examples include giving a team member access to a channel you’ve recently connected, removing a user’s access to specific channels, or adding Admin access for a user.</p>
<p>This guide will help you edit all of your user permissions, but if you're starting from scratch, visit our article: <a href="https://buffer.helpscoutdocs.com/article/670-adding-users-and-setting-up-permissions-in-your-organization" target="_self">Adding users and setting up permissions in your organization</a>.</p>
<p>In this article:</p>
<ol>
 <li><a href="#h_01EJGJ6YRQ9HVZZX6YW56VBBHB" target="_self" rel="undefined">Editing a user’s channel permissions</a></li>
 <li><a href="#h_01EJGJ77Z2R540Z0B53VZPWKEA" target="_self" rel="undefined">Adding or removing a user on a channel</a></li>
 <li><a href="#h_01EJGJ7EXZHPJDZZF3J1PQ8TKY" target="_self">Assigning permissions in bulk</a></li>
 <li><a href="#h_01EJGJ7PWG9MKCMJWHZJQ0A8FS" target="_self">Editing a user’s admin privileges</a></li>
</ol>
<h2 id="h_01EJGJ6YRQ9HVZZX6YW56VBBHB">Editing a user’s channel permissions</h2>
<p>Please follow these steps to edit your user’s channel permissions.</p>
<ol>
 <li>Click on your profile avatar at the top right of your dashboard and then click<i><b> Team</b></i>.</li>
 <li>From your organization dashboard, click on the user you'd like to work with.</li>
 <li>To add or change permissions the user has access to for a specific channel, use the drop down to change to the desired permissions.</li>
</ol>
<p><img src="//d33v4339jhl8k0.cloudfront.net/docs/assets/64cb6cf86d4a3e191f237e66/images/67ae3cc8769d1a4e1f4ed971/file-a0wqffwgYB.jpg" alt="Team member channel permission toggle" style="max-width: 100%; "></p>
<h2 id="h_01EJGJ77Z2R540Z0B53VZPWKEA">Adding or removing a user on a channel</h2>
<p>To add a user to a channel, type the channel name in the box below and then choose the access level you’d like to give.</p>
<p><img src="//d33v4339jhl8k0.cloudfront.net/docs/assets/64cb6cf86d4a3e191f237e66/images/67ae3e9ec64ec2266f232b8a/file-iG3YLBAgPK.png" alt="Assign channels in bulk" style="width: 597.979px; "></p>
<p>Or, to remove their access to the channel, click&nbsp;<em><strong>Remove</strong></em>.</p>
<p><img src="//d33v4339jhl8k0.cloudfront.net/docs/assets/64cb6cf86d4a3e191f237e66/images/67ae3d2029a45508db9a3914/file-gqxAdBS5aF.png" alt="Remove team member channel access" style="width: 648.984px; "></p>
<h2 id="h_01EJGJ7EXZHPJDZZF3J1PQ8TKY">Assigning permissions in bulk</h2>
<p>If you are managing lots of channels, you may prefer to assign permissions in bulk. This can be done by visiting the user's profile and clicking the&nbsp;<em><strong>assign in bulk</strong></em>&nbsp;link.</p>
<p><img src="//d33v4339jhl8k0.cloudfront.net/docs/assets/64cb6cf86d4a3e191f237e66/images/67ae3d461dbfa0582db2818e/file-A74bWJGFSA.png" alt="Team member assign in bulk" style="max-width: 100%; "></p>
<p>From there, select the channels the user should have access to on the left hand side, and then choose the level of access they should have. Finally, click&nbsp;<em><strong>Save Changes</strong></em>.</p>
<p><img src="//d33v4339jhl8k0.cloudfront.net/docs/assets/64cb6cf86d4a3e191f237e66/images/67ae3d6cc64ec2266f232b7e/file-t4vVJlJTVN.png" alt="Save permissions changes in bulk" style="max-width: 100%; "></p>
<h2 id="h_01EJGJ7PWG9MKCMJWHZJQ0A8FS">Editing a user’s admin privileges</h2>
<p>To add or remove admin access from a user in your organization, please follow these steps.</p>
<ol>
 <li>From your organization dashboard, click on the user you'd like to work with.</li>
 <li>Click&nbsp;<strong><em>Edit User</em></strong>.</li>
</ol>
<p><img src="//d33v4339jhl8k0.cloudfront.net/docs/assets/64cb6cf86d4a3e191f237e66/images/67ae3d8fb412812df3235e40/file-0LrTuTnuRO.png" alt="Edit user button in team member settings" style="max-width: 100%; "></p>
<ol start="3">
 <li>Toggle&nbsp;<strong><em>Admin Access</em></strong>&nbsp;to the access you would like to give the user and click <i><b>Save Changes</b></i>.</li>
</ol>
<p><img src="//d33v4339jhl8k0.cloudfront.net/docs/assets/64cb6cf86d4a3e191f237e66/images/67ae3e03769d1a4e1f4ed979/file-FUB3md8K0h.jpg" alt="Buffer User Details screen with an arrow pointed to the Admin Access Toggle" style="max-width: 100%; "></p>
<p>To completely remove a user from your organization, see the guide: <a href="https://buffer.helpscoutdocs.com/article/672-removing-a-user-from-your-organization" target="_blank">Removing a user from your organization</a>.</p>
