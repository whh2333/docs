# How to avoid being logged out when using the Buffer browser extension in Safari <a href="javascript:window.print()" class="printArticle" title="Print this article" aria-label="Print this Article"><i class="icon-print"></i></a> <p>If you're using Safari and find that you're frequently logged out of Buffer's [browser extension](https://buffer.helpscoutdocs.com/article/653-buffer-browser-extension), changing your privacy settings can help. Please follow the steps below.</p> - In Safari, click ***Safari*** and then ***Settings***.</li> - Select the ***Privacy*** tab and then un-check the "Prevent cross-site tracking" box.</li> - Restart Safari and you should be all set!
