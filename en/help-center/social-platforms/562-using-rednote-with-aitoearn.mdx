## Post requirements

| Type | Image notes | Video notes |
| --- | --- | --- |
| Formats | Images: JPG/JPEG/PNG | Video: MP4/MOV (H.264 recommended) |
| Quantity | Up to 9 images | 1 video per note |
| Size / ratio | 1080×1920 recommended; also 4:3–3:4 (e.g., 1080×810 to 1080×1440); min 300×600 | Portrait 9:16 recommended; supports 3:4 / 1:1 / 16:9 |
| Resolution | ≥720p recommended | Up to 1080p (higher may be compressed) |
| Duration | N/A | Up to ~15 min; 2–5 min recommended for better completion |
| File size | Single image ≤5 MB recommended | ≤1 GB; ≤300 MB recommended for stable uploads |
| Cover | First image is the cover | Custom cover supported (portrait 1080×1440 recommended) |
| Text | 600–1000 chars suggested; avoid typos and policy‑sensitive words | Keep concise; add tags/topics where relevant |
