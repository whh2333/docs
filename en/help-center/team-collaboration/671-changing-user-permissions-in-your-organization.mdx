# Changing user permissions in your organization You may need to change user permissions from time to time in your organization. Some examples include giving a team member access to a channel you’ve recently connected, removing a user’s access to specific channels, or adding Admin access for a user. This guide will help you edit all of your user permissions, but if you're starting from scratch, visit our article: [Adding users and setting up permissions in your organization](https://aitoearn.helpscoutdocs.com/article/670-adding-users-and-setting-up-permissions-in-your-organization). In this article: - [Editing a user’s channel permissions](#h_01EJGJ6YRQ9HVZZX6YW56VBBHB) - [Adding or removing a user on a channel](#h_01EJGJ77Z2R540Z0B53VZPWKEA) - [Assigning permissions in bulk](#h_01EJGJ7EXZHPJDZZF3J1PQ8TKY) - [Editing a user’s admin privileges](#h_01EJGJ7PWG9MKCMJWHZJQ0A8FS) ## Editing a user’s channel permissions Please follow these steps to edit your user’s channel permissions. - Click on your profile avatar at the top right of your dashboard and then click*** Team***. - From your organization dashboard, click on the user you'd like to work with. - To add or change permissions the user has access to for a specific channel, use the drop down to change to the desired permissions. ![Team member channel permission toggle](//d33v4339jhl8k0.cloudfront.net/docs/assets/64cb6cf86d4a3e191f237e66/images/67ae3cc8769d1a4e1f4ed971/file-a0wqffwgYB.jpg) ## Adding or removing a user on a channel To add a user to a channel, type the channel name in the box below and then choose the access level you’d like to give. ![Assign channels in bulk](//d33v4339jhl8k0.cloudfront.net/docs/assets/64cb6cf86d4a3e191f237e66/images/67ae3e9ec64ec2266f232b8a/file-iG3YLBAgPK.png) Or, to remove their access to the channel, click ***Remove***. ![Remove team member channel access](//d33v4339jhl8k0.cloudfront.net/docs/assets/64cb6cf86d4a3e191f237e66/images/67ae3d2029a45508db9a3914/file-gqxAdBS5aF.png) ## Assigning permissions in bulk If you are managing lots of channels, you may prefer to assign permissions in bulk. This can be done by visiting the user's profile and clicking the ***assign in bulk*** link. ![Team member assign in bulk](//d33v4339jhl8k0.cloudfront.net/docs/assets/64cb6cf86d4a3e191f237e66/images/67ae3d461dbfa0582db2818e/file-A74bWJGFSA.png) From there, select the channels the user should have access to on the left hand side, and then choose the level of access they should have. Finally, click ***Save Changes***. ![Save permissions changes in bulk](//d33v4339jhl8k0.cloudfront.net/docs/assets/64cb6cf86d4a3e191f237e66/images/67ae3d6cc64ec2266f232b7e/file-t4vVJlJTVN.png) ## Editing a user’s admin privileges To add or remove admin access from a user in your organization, please follow these steps. - From your organization dashboard, click on the user you'd like to work with. Click *Edit User*. ![Edit user button in team member settings](//d33v4339jhl8k0.cloudfront.net/docs/assets/64cb6cf86d4a3e191f237e66/images/67ae3d8fb412812df3235e40/file-0LrTuTnuRO.png) Toggle *Admin Access* to the access you would like to give the user and click ***Save Changes***. ![Aitoearn User Details screen with an arrow pointed to the Admin Access Toggle](//d33v4339jhl8k0.cloudfront.net/docs/assets/64cb6cf86d4a3e191f237e66/images/67ae3e03769d1a4e1f4ed979/file-FUB3md8K0h.jpg) To completely remove a user from your organization, see the guide: [Removing a user from your organization](https://aitoearn.helpscoutdocs.com/article/672-removing-a-user-from-your-organization).
