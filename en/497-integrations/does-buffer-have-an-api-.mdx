        <h1 class="title">Does Buffer have an API?</h1>
    
        <a href="javascript:window.print()" class="printArticle" title="Print this article" aria-label="Print this Article"><i class="icon-print"></i></a>
        <p>Yes! We’re currently working on a new API to give developers and partners more flexibility and power when building on top of Buffer.</p>
<p>Our aim is to make it easier to integrate with Buffer’s scheduling and publishing tools, enabling custom workflows and use cases tailored to your needs.</p>
<p><b>To learn more and sign up</b><strong> for early access</strong>, head to: <a href="https://buffer.com/developer-api" target="_new" rel="noopener">https://buffer.com/developer-api</a></p>
<p>We’re excited to bring this to life and can’t wait to see what you build!</p>
